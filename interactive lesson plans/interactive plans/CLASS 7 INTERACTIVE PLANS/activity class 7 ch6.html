Coin Problem

Lakpa has an odd number of ₹1 coins, an odd number of ₹5 coins and an even number of ₹10 coins in his piggy bank. He calculated the total and got ₹205. Did he make a mistake?

Hint: Think about the parity of each coin type and how they add up.

Think about the parity of each coin type:

  • ₹1 coins: odd number × ₹1 (odd) = odd total
  • ₹5 coins: odd number × ₹5 (odd) = odd total
  • ₹10 coins: even number × ₹10 (even) = even total

Sum: odd + odd + even = even

But ₹205 is odd. So, is this possible?
