Problem Statement

Maximize: z = 6x + 3y

Subject to the constraints:

4x + y ≥ 80
3x + 2y ≤ 150
x + 5y ≥ 115
x ≥ 0, y ≥ 0

Solution Approach

To solve this linear programming problem graphically, we'll:

  1. Plot the constraint inequalities on a graph
  2. Identify the feasible region that satisfies all constraints
  3. Find the corner points of the feasible region
  4. Evaluate the objective function at each corner point
  5. Determine the point that maximizes the objective function

Optimal Solution

The maximum value of z = 6x + 3y is 285.

This maximum value occurs at the point (40, 15).

Key Observations

This problem demonstrates a typical linear programming problem where:

  • The feasible region is bounded
  • The objective function has a unique optimal solution
  • The optimal solution occurs at a vertex of the feasible region

Problem Statement

Minimize: z = x + 2y

Subject to the constraints:

2x + y ≥ 3
x + 2y ≥ 6
x ≥ 0, y ≥ 0

Solution Approach

To solve this linear programming problem graphically, we'll:

  1. Plot the constraint inequalities on a graph
  2. Identify the feasible region that satisfies all constraints
  3. Find the corner points of the feasible region
  4. Evaluate the objective function at each corner point
  5. Determine the point that minimizes the objective function

Corner Points Analysis

The corner points of the feasible region are:

Corner Point (x, y) Calculation of z = x + 2y Value of z
(0, 3) 0 + 2×3 6
(6, 0) 6 + 2×0 6

Optimal Solution

The minimum value of z = x + 2y is 6.

This minimum value occurs at multiple points along the line segment connecting (0, 3) and (6, 0).

Any point on the line x + 2y = 6 that lies within the feasible region will yield the minimum value of z = 6.

Key Observations

This problem demonstrates a special case in linear programming where:

  • The feasible region is unbounded
  • The objective function has the same optimal value at multiple points
  • The optimal solution occurs along an edge of the feasible region rather than at a single vertex
