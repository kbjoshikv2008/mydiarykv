27. The probabilities of a specific problem being solved independently by A and B are \( \frac{1}{2} \) and \( \frac{1}{3} \) respectively. If both try to solve the problem independently, find the probability that (i) The problem is solved (ii) Exactly one of them solves the problem.

Let E₁ = event that A solves the problem, E₂ = event that B solves the problem

P(E₁) = 1/2, P(E₂) = 1/3

Since events are independent, P(E₁ ∩ E₂) = 1/6

(i) P(problem is solved) = P(E₁ ∪ E₂) = P(E₁) + P(E₂) - P(E₁ ∩ E₂) = 1/2 + 1/3 - 1/6 = 2/3

(ii) P(exactly one solves) = P(E₁)P(not E₂) + P(E₂)P(not E₁) = (1/2)(2/3) + (1/3)(1/2) = 1/3 + 1/6 = 1/2

33. Let \( A = \{1, 2, 3, 4, 5, 6, 7, 8, 9\} \) and R be a relation in \( A \times A \), defined by (a, b) R (c, d) \( \Leftrightarrow \) a+d= b+c for all (a,b) and (c, d) \( \in A \times A \). Prove that R is an equivalence relation. Also find the equivalence class of (2, 5).

OR

Consider f: R→[9,∞) given by f(x)=5x²+6x-9. Prove that f is bijective.

Main:

Reflexive: (a,b)R(a,b) since a+b = b+a

Symmetric: If (a,b)R(c,d) then a+d = b+c ⇒ c+b = d+a ⇒ (c,d)R(a,b)

Transitive: If (a,b)R(c,d) and (c,d)R(e,f) then a+d=b+c and c+f=d+e

Adding: a+d+c+f = b+c+d+e ⇒ a+f = b+e ⇒ (a,b)R(e,f)

Equivalence class of (2,5) = {(a,b): 2+b = 5+a ⇒ b-a=3}

= {(1,4), (2,5), (3,6), (4,7), (5,8), (6,9)}

OR: f(x)=5x²+6x-9 = 5(x² + 6/5 x) - 9 = 5[(x+3/5)² - 9/25] - 9

= 5(x+3/5)² - 9/5 - 9 = 5(x+3/5)² - 54/5

Minimum value = -54/5 when x = -3/5

Since coefficient of x² is positive, f(x) is increasing for x > -3/5 and decreasing for x < -3/5

f is one-one and onto, hence bijective.

38. Mahindra Tractors is India's leading farm equipment manufacturer. It is the largest tractor selling factory in the world. This factory has two machine A and B. Past record shows that machine A produced 60% and machine B produced 40% of the output(tractors). Further 2% of the tractors produced by machine A and 1% produced by machine B were defective. All the tractors are put into one big store hall and one tractor is chosen at random.

(i) Find the total probability of chosen tractor (at random) is defective.

(ii) If in random choosing, chosen tractor is defective, then find the probability that the chosen tractor is produced by machine 'A'

Let E₁ = event that tractor is from machine A, E₂ = event that tractor is from machine B

P(E₁) = 60/100 = 3/5, P(E₂) = 40/100 = 2/5

P(defective|E₁) = 2/100 = 1/50, P(defective|E₂) = 1/100

(i) P(defective) = P(E₁)P(defective|E₁) + P(E₂)P(defective|E₂)

= (3/5)(1/50) + (2/5)(1/100) = 3/250 + 2/500 = 6/500 + 2/500 = 8/500 = 2/125

(ii) P(E₁|defective) = \( \frac{P(E₁)P(defective|E₁)}{P(defective)} = \frac{(3/5)(1/50)}{2/125} = \frac{3/250}{2/125} = \frac{3}{250} \times \frac{125}{2} = \frac{3}{4} \)
