Key Concepts

Definitions, types, and rules for solving linear inequalities.

What is an Inequality?

Two real numbers or algebraic expressions related by <, >, ≤ or ≥ form an inequality.

e.g.   3 < 5  ·  x > 2  ·  3 ≤ x < 5  ·  ax + b ≤ 0
Numerical: 7 > 5 Literal: x < 5 Double: 3 ≤ x < 5

Types of Inequalities

Strict <   > e.g. ax + b < 0
Slack ≤   ≥ e.g. ax + b ≤ 0
Linear in one variable ax + b < 0   (a ≠ 0)
Linear in two variables ax + by < c   (a,b ≠ 0)

Quadratic inequalities (e.g. ax²+bx+c ≤ 0) are not linear — we focus only on linear.

Rules for Solving Inequalities

Rule 1 Addition / Subtraction Same number can be added or subtracted from both sides. No sign change.
Rule 2 Multiplication / Division By positive number → sign unchanged.
By negative number → sign reverses.
Remember: \(-3 < -2\) but \(3 > 2\). Multiplying by \(-1\) flips the inequality.

Solution Set

The set of all values of the variable that make the inequality true is called the solution set.

e.g. \( 30x < 200 \)  →  \( x < \frac{20}{3} \)  →  solution set \( (-\infty, \frac{20}{3}) \)

When \(x\) is natural, the solution set is \(\{1,2,3,4,5,6\}\).

Graphical Representation

x < a → open circle, dark line to left
x ≤ a → closed circle, dark line to left
x > a → open circle, dark line to right
x ≥ a → closed circle, dark line to right

Worked Examples

Example 1: Solve \(30x < 200\)

(i) \(x\) is natural   (ii) \(x\) is integer

\(30x < 200 \;\Rightarrow\; x < \frac{20}{3} \approx 6.67\)

(i) Natural numbers: \(\{1,2,3,4,5,6\}\)
(ii) Integers: \(\{\dots, -3,-2,-1,0,1,2,3,4,5,6\}\)
Example 2: Solve \(5x - 3 < 3x + 1\)

(i) \(x\) integer   (ii) \(x\) real

\(5x - 3 < 3x + 1\)
\(\Rightarrow 5x - 3x < 1 + 3\)
\(\Rightarrow 2x < 4 \;\Rightarrow\; x < 2\)

(i) Integers: \(\{\dots, -3,-2,-1,0,1\}\)
(ii) Real: \(x \in (-\infty, 2)\)
Example 3: Solve \(4x + 3 < 6x + 7\)
\(4x + 3 < 6x + 7\)
\(\Rightarrow 4x - 6x < 7 - 3\)
\(\Rightarrow -2x < 4\)
Divide by \(-2\) → sign reverses → \(x > -2\) Solution set: \((-2, \infty)\)
Example 4: Solve \(\frac{5-2x}{3} \le \frac{x}{6} - 5\)
Multiply by 6: \(2(5-2x) \le x - 30\)
\(\Rightarrow 10 - 4x \le x - 30\)
\(\Rightarrow -5x \le -40\)
Divide by \(-5\) → sign reverses: \(x \ge 8\)
Solution set: \([8, \infty)\)
Example 5: Solve \(-8 \le 5x - 3 < 7\)
\(-8 \le 5x - 3 < 7\)
Add 3: \(-5 \le 5x < 10\)
Divide by 5: \(-1 \le x < 2\)
Solution set: \([-1, 2)\)
Example 6: System \(3x-7 < 5+x\) and \(11-5x \le 1\)
From (1): \(3x - 7 < 5 + x \Rightarrow 2x < 12 \Rightarrow x < 6\)
From (2): \(11 - 5x \le 1 \Rightarrow -5x \le -10 \Rightarrow x \ge 2\)
Intersection: \(2 \le x < 6\)  →  \([2, 6)\)
Example 7: Marks average at least 60

First two tests: 62 and 48. Find min marks in third for average ≥ 60.

Let \(x\) be third marks.
\(\frac{62+48+x}{3} \ge 60\)
\(\Rightarrow 110 + x \ge 180 \Rightarrow x \ge 70\)
Minimum marks = 70
Example 8: Consecutive odd > 10, sum < 40
Let numbers be \(x\) and \(x+2\) (both odd).
\(x > 10\) and \(x + (x+2) < 40 \Rightarrow 2x+2 < 40 \Rightarrow x < 19\)
So \(10 < x < 19\). Odd values: 11, 13, 15, 17.
Pairs: (11,13), (13,15), (15,17), (17,19)

Real‑World Applications

Inequalities in science, economics, and everyday life.

Temperature Range

A solution must be kept between \(30^\circ\)C and \(35^\circ\)C. Using \(\mathrm{C} = \frac{5}{9}(\mathrm{F}-32)\), find the Fahrenheit range.

\(30 < \frac{5}{9}(F-32) < 35\)
Multiply by \(\frac{9}{5}\): \(54 < F-32 < 63\)
Add 32: \(86 < F < 95\)
Range: \(86^\circ\)F to \(95^\circ\)F

Acid Mixture

A manufacturer has 600 L of 12% acid. How many litres of 30% acid must be added so the mixture is more than 15% but less than 18% acid?

Let \(x\) = litres of 30% acid added.
Total = \(x+600\).
Acid: \(0.30x + 0.12(600) = 0.30x + 72\)
Need: \(0.15(x+600) < 0.30x+72 < 0.18(x+600)\)
Solve: \(15x+7200 < 30x+7200 \Rightarrow 15x > 1800 \Rightarrow x > 120\)
And \(30x+7200 < 18x+10800 \Rightarrow 12x < 3600 \Rightarrow x < 300\)
\(120 < x < 300\) litres

Board Cutting

A 91 cm board is cut into 3 pieces. Second is 3 cm longer than shortest, third is twice the shortest. Third must be at least 5 cm longer than second. Find possible lengths of shortest piece.

Let shortest = \(x\). Then pieces: \(x,\;x+3,\;2x\).
Total: \(x+(x+3)+2x \le 91 \Rightarrow 4x+3 \le 91 \Rightarrow x \le 22\)
Third ≥ second + 5: \(2x \ge (x+3)+5 \Rightarrow 2x \ge x+8 \Rightarrow x \ge 8\)
\(8 \le x \le 22\) cm

IQ Range

\(\mathrm{IQ} = \frac{\mathrm{MA}}{\mathrm{CA}}\times 100\). For 12‑year‑olds (\(CA=12\)), if \(80 \le IQ \le 140\), find range of mental age (MA).

\(80 \le \frac{MA}{12}\times 100 \le 140\)
Divide by 100: \(0.8 \le \frac{MA}{12} \le 1.4\)
Multiply by 12: \(9.6 \le MA \le 16.8\)
Mental age between 9.6 and 16.8 years.
