Linear Programming

Introduction to Linear Programming

Linear programming is a mathematical method to achieve the best outcome (such as maximum profit or lowest cost) in a mathematical model whose requirements are represented by linear relationships.

Example: Furniture Dealer Problem

A furniture dealer deals in only two items–tables and chairs. He has Rs 50,000 to invest and has storage space of at most 60 pieces. A table costs Rs 2500 and a chair Rs 500. He estimates a profit of Rs 250 from each table and Rs 75 from each chair. How many tables and chairs should he buy to maximize his profit?

Mathematical Formulation

Let \( x \) be the number of tables and \( y \) be the number of chairs.

Objective Function (to maximize profit):

Maximize \( Z = 250x + 75y \)

Constraints:

\( 5x + y \leq 100 \)   (Investment constraint: \( 2500x + 500y \leq 50000 \))
\( x + y \leq 60 \)   (Storage constraint)
\( x \geq 0, y \geq 0 \)   (Non-negative constraints)

Graphical Method

The graphical method involves plotting the constraints on a graph to identify the feasible region, which contains all possible solutions that satisfy the constraints.

The optimal solution lies at one of the corner points of the feasible region. We evaluate the objective function at each corner point to find the maximum or minimum value.

Corner Points and Objective Values

Corner Point (x, y) Calculation of Z = 250x + 75y Value of Z (Profit)
(0, 0) 250(0) + 75(0) 0
(20, 0) 250(20) + 75(0) 5000
(0, 60) 250(0) + 75(60) 4500
(10, 50) 250(10) + 75(50) 6250

The maximum profit of Rs 6250 occurs at (10, 50), meaning the dealer should buy 10 tables and 50 chairs.

Key Concepts

Feasible Region: The set of all points that satisfy all constraints.

Objective Function: The linear function to be maximized or minimized.

Corner Point Principle: The optimal solution lies at a corner point of the feasible region.
