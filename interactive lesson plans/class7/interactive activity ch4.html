🔥 Activity 2: Matchstick L's
Each L needs 2 matchsticks. Number of matchsticks = \(2 \times n\) where \(n\) = number of Ls.
Example: 5 Ls → \(5 \times 2 = 10\) matchsticks.
🔢 How many matchsticks are needed for 12 Ls? ( \(12 \times 2\) )
📝 Write the algebraic expression for \(n\) Ls.
