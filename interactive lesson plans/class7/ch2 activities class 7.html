🔍 2. Terms & Brackets — Order matters!

🧩 TERMS: Parts separated by + (after converting subtraction to adding inverse).
Example: 30 + 5×4 → terms: 30 and 5×4. Evaluate each term first then add.
🧮 Brackets force order: 30 + (5×4) = 50 but without brackets we use term method.
✏️ Evaluate by identifying terms (Order of operations)

Find value of: 28 − 7 + 8

Find value of: 39 − 2×6 + 11 (× first)

48 − 10×2 + 16÷2 (multiply & divide first)

💡 Did you know? 30+5×4 without context gave 140 (wrong) vs 50 (correct). Terms save us!

📌 Quick rules to remember
• Expression value: evaluate each term first.
• Brackets first > then ×÷ > then +−
• -(a+b) = -a -b     -(a-b) = -a + b
• Commutative: 5+3 = 3+5
• Anything + its inverse = 0
