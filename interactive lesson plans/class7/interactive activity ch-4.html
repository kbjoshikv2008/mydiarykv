🧩 4.1 The Notion of Letter‑Numbers
Letter‑numbers are letters (like \(a, s, n, x\)) that stand for numbers. Expressions containing them are called algebraic expressions. They let us write general rules in a short, clear way.
📖 Example 1 – Age Relation
Shabnam is 3 years older than Aftab.
If Aftab's age = \(a\) years, then Shabnam's age = \(a + 3\).
If Shabnam's age = \(s\) years, then Aftab's age = \(s - 3\).
1 Write the algebraic expression for Aftab's age when Shabnam's age is \(s\).
2 If Aftab is 23 years old, use \(a+3\) to find Shabnam's age.
💡 Key idea: The expression \(a+3\) works for any value of \(a\). That's the power of algebra — one formula for all cases!

📦 4.4 Simplification – Like & Unlike Terms
Like terms have the same letter‑numbers (e.g. \(5c, 3c, 10c\)).
Unlike terms have different letter‑numbers (e.g. \(18c, 11d\)).
Like terms can be added or subtracted to simplify an expression.
✔️ \(5c + 3c + 10c = 18c\)    ✔️ \(12n - 4n = 8n\)
🔹 \((40x + 75y) - (6x + 10y) = 34x + 65y\)
🔹 \(4(x+y) - y = 4x + 3y\)
14 Simplify: \(7p + 3p - 2p + 5q\)
