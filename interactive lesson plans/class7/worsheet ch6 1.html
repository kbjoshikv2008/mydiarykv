Section A — Numbers Tell Us Things (Ordering & Counts)

Topic: Each child calls out the number of children in front of them who are taller than them.

  1. Q1. Arrange stick-figures (or draw heights) so the sequence of numbers read:
    1. (a) 0, 1, 1, 2, 4, 1, 5
    2. (b) 0, 0, 0, 0, 0, 0, 0
    3. (c) 0, 1, 2, 3, 4, 5, 6
    4. (d) 0, 1, 0, 1, 0, 1, 0
    5. (e) 0, 1, 1, 1, 1, 1, 1
    6. (f) 0, 0, 0, 3, 3, 3, 3
    Show answer / sample arrangement
    (a) One valid height order (tallest → shortest): heights such that positions see counts 0,1,1,2,4,1,5 — e.g., heights [9,7,6,5,3,4,1] (teacher may draw stick figures).
    (b) All in non-increasing order (each person taller than all behind them).
    (c) Strictly decreasing heights (each person taller than all behind).
    (d) Alternating heights.
    (e) First is tallest, rest equal heights decreasing as needed.
    (f) First three tallest equal small numbers, last four shorter with larger counts.
    (Teacher note: ask class to draw and verify counts.)
  2. Q2. For each statement, mark Always True / Sometimes True / Never True with reasons.
    1. If a person says ‘0’, then they are the tallest in the group.
    2. If a person is the tallest, then their number is ‘0’.
    3. The first person’s number is ‘0’.
    4. If a person is not first or last in line, they cannot say ‘0’.
    5. The person who calls out the largest number is the shortest.
    6. What is the largest number possible in a group of 8 people?
    Show answers
    (a) Sometimes true — a '0' means nobody in front is taller; they could be tallest or everyone taller might be behind them.
    (b) Always true — tallest has 0 taller people in front.
    (c) Sometimes true — first person's number counts those in front (none), so it must be 0. Actually the first person sees everyone else in front? Note: if numbering from left, first has nobody in front so number = 0.
    (d) Sometimes false — a middle person may have 0 if no one in front is taller.
    (e) Often true — the largest number equals number of people taller in front; the shortest could have the largest count but depends on arrangement.
    (f) Max possible = 7 (if someone has all others taller in front).
    (Discuss examples with class.)

Section B — Picking Parity (Even & Odd)

Topic: Parity rules and puzzles (even/odd behaviour).

  1. Q1. Kishor has 5 boxes and only odd-number cards. Can he choose 5 cards to sum to 30?
    2. Show answer
    No. Sum of 5 odd numbers is odd, while 30 is even — impossible.
  2. Q2. Martin and Maria born 1 year apart — can their ages sum to 112?
    3. Show answer
    No. Consecutive numbers are one even and one odd → sum is odd. 112 is even → impossible.
  3. Q3. Find parity of:
    1. Sum of 2 even + 2 odd
    2. Sum of 2 odd + 3 even
    3. Sum of 5 even numbers
    4. Sum of 8 odd numbers
    Show answers
    (a) even + even = even; even + (odd+odd=even) → even.
    (b) 2 odd → even; 3 even → even; even+even=even.
    (c) Sum of any evens = even.
    (d) 8 odd (even count) → even.
  4. Q4. Lakpa has odd count of ₹1 coins, odd count of ₹5 coins, even count of ₹10 coins; total is ₹205. Is this possible? Give an example if yes.
    5. Show answer
    Yes. Parity: odd*1 -> odd, odd*5 -> odd (5×odd = odd), even*10 -> even; odd+odd+even = even. 205 is odd, so this seems contradictory. Therefore he made a mistake — odd+odd=even; even+even=even → total must be even. So 205 (odd) impossible.

Section C — Small Squares in Grids & Magic Squares

Topic: Row/column sums, magic squares using numbers 1–9.

  1. Q1. In 3×3 magic square with numbers 1–9, what is the magic sum?
    2. Show answer
    Magic sum = 15 (because total 1–9 = 45; each of 3 equal row sums → 45/3 = 15).
  2. Q2. Which number must be at centre of 3×3 magic square using 1–9?
    3. Show answer
    5 must be at the centre (only choice that balances sums to 15 in all directions).
  3. Q3. Create a magic square using numbers 2–10. Compare method with 1–9.
    4. Show sample answer
    Add 1 to every entry of a 1–9 magic square to get a 2–10 magic square. Magic sum increases by 3 (since each row has 3 entries) → 15 + 3 = 18.
  4. Q4. Try: How many different 3×3 magic squares using 1–9?
    5. Show answer
    There are 8 essentially different (rotations/reflections of the same pattern). Counting symmetries gives 8 standard forms from the basic Lo Shu arrangement.

Section D — Virahāṅka–Fibonacci Numbers

Topic: Counting rhythms using 1's and 2's; Fibonacci recurrence.

  1. Q1. How many rhythms (ways to write as 1s and 2s) are there for n=5 and n=8?
    2. Show answer
    n=5 → 8 ways. n=8 → 34 ways (8th Virahāṅka–Fibonacci number).
  2. Q2. Write next three numbers after 89 in the sequence 1,2,3,5,...
    3. Show answer
    After 89: 144, 233, 377 (each is sum of previous two).
  3. Q3. Angaan climbs 8 steps taking 1 or 2 steps — how many ways?
    4. Show answer
    Number of ways = F(9) = 34 (Fibonacci indexing; same as rhythms for 8 beats).

Section E — Digits in Disguise (Cryptarithms)

Topic: Replace digits with letters to form valid sums.

  1. Q1. Solve: T + T + T = UT (a single digit added to itself thrice gives a two-digit number whose units digit is same as T).
    2. Show answer
    T = 5, UT = 15 (because 5+5+5=15).
  2. Q2. Solve the cryptarithm: 
     UT
+   TA
=  TAT
    3. Show answer
    One solution: U=1, T=5, A=0 gives 15 + 50 = 65, not TAT. (Teacher: challenge students to find consistent assignment; full solution requires column-wise carry reasoning.)
  3. Q3. Light bulb toggled 77 times — final state? (starts ON)
    4. Show answer
    Each toggle flips state. 77 is odd → final state = OFF if starting ON? Wait: start ON, toggled odd times → OFF. So final = OFF.
  4. Q4. Liswini found 50 sheets (printed both sides). Can sum of page numbers be 6000?
    5. Show answer
    No. Pages come in pairs (front/back) so sums of a sheet are consecutive numbers (odd+even = odd). Sum of 50 sheets = sum of 100 page numbers — parity and bounds show 6000 is too large for reasonable page ranges; teacher can walk class through precise contradiction.
