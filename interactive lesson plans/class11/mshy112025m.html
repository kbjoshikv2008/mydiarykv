1. Which of the following is an empty set?

a) {x: |x| < 4, x ∈ N}
b) {x: |x| < -4, x ∈ N}
c) {x: x² = 4, x ∈ N}
d) {x: x² = 4, x ∈ R}

Answer: b) {x: |x| < -4, x ∈ N}

Explanation: The absolute value |x| is always non-negative, so |x| < -4 has no solution.

2. The domain of the function f given by f(x) = (x²+2x+1)/(x²-x-6)

a) R - {3, -2}
b) R - {-3, 2}
c) R - [3, -2]
d) R - (3, -2)

Answer: a) R - {3, -2}

Explanation: The denominator x²-x-6 = (x-3)(x+2) must not be zero, so x ≠ 3 and x ≠ -2.

3. The first term of a G.P is 1. The sum of the 3rd and 5th terms is 90. Then the common ratio is

a) 1
b) 2
c) 3
d) 4

Answer: c) 3

Explanation: Let a = 1 (first term), r = common ratio
3rd term = ar² = r²
5th term = ar⁴ = r⁴
Given: r² + r⁴ = 90
r⁴ + r² - 90 = 0
Let y = r², then y² + y - 90 = 0
(y+10)(y-9) = 0
y = 9 or y = -10 (rejected as r² cannot be negative)
So r² = 9 ⇒ r = 3 or -3

4. 8th term from the end of the sequence 3, 6, 12, …… 25th term is

a) 393216
b) 393206
c) 313216
d) 303216

Answer: a) 393216

Explanation: This is a GP with a = 3, r = 2
25th term from beginning = ar²⁴ = 3 × 2²⁴
8th term from end means 18th term from beginning (25-8+1=18)
18th term = ar¹⁷ = 3 × 2¹⁷ = 3 × 131072 = 393216

5. The value of x so that the line through (3, x) and (2, 7) is parallel to the line through (-1, 4) and (0, 6)

a) 3
b) 6
c) 9
d) -9

Answer: c) 9

Explanation: Slope of line through (3, x) and (2, 7) = (7-x)/(2-3) = (7-x)/(-1) = x-7
Slope of line through (-1, 4) and (0, 6) = (6-4)/(0-(-1)) = 2/1 = 2
For parallel lines, slopes are equal: x-7 = 2 ⇒ x = 9

6. If sinθ + cosθ = 1, then the value of sin2θ is equal to

a) 1
b) 1/2
c) 0
d) -1

Answer: c) 0

Explanation: sinθ + cosθ = 1
Squaring both sides: (sinθ + cosθ)² = 1²
sin²θ + cos²θ + 2sinθcosθ = 1
1 + sin2θ = 1
sin2θ = 0

9. If 18Cr = 18Cr+10 then 16Cr equals to

a) 820
b) 420
c) 116
d) none

Answer: d) none

Explanation: Using property: nCr = nCn-r
Given: 18Cr = 18Cr+10
So either r = r+10 (impossible) or r + (r+10) = 18
2r + 10 = 18 ⇒ 2r = 8 ⇒ r = 4
Then 16Cr = 16C4 = 1820

a) 50
b) 52
c) 26

Answer: c) 26

This gives 26 terms (from exponent 51 down to 1 in steps of 2)

17. If the parabola y² = 4ax passes through the point (3, 2), then the length of its latus rectum is

18. The equation x²+y²+2x-4y+5=0 represents

a) a point
b) a circle with non-zero radius
c) a circle with zero radius
d) a pair of straight line

Answer: a) a point

Explanation: x²+y²+2x-4y+5=0

20. Assertion (A): Sum of the series -3/4, 3/16, -3/64, ... is -2/3.
Reason (R): If a₁, a₂, a₃ are in GP, then a₂/a₁ = a₃/a₂.

Answer: b) Both A and R are true but R is not the correct explanation of A

Explanation: Assertion A: GP with a = -3/4, r = -1/4
Sum = a/(1-r) = (-3/4)/(1+1/4) = (-3/4)/(5/4) = -3/5, not -2/3
So A is false
Reason R is true: In a GP, the ratio between consecutive terms is constant
Since A is false and R is true, the answer is (d) A is false but R is true

22. Find the lengths of major and minor axes, coordinates of foci, vertices and the eccentricity: 3x² + 2y² = 6

Solution:

Divide by 6: x²/2 + y²/3 = 1
This is vertical ellipse with a² = 3, b² = 2
Major axis length = 2√3, Minor axis length = 2√2
c² = a² - b² = 3 - 2 = 1 ⇒ c = 1
Foci: (0, ±1)
Vertices: (0, ±√3)
Eccentricity: e = c/a = 1/√3

SECTION C: Short Answer Type Questions

26. If tan x = -5/12, x lies in quadrant II. Find sin(x/2), cos(x/2) and tan(x/2).

Solution:

In QII: sin x > 0, cos x < 0
sin x = 5/13, cos x = -12/13
sin(x/2) = √[(1-cos x)/2] = √[(1+12/13)/2] = √[(25/13)/2] = √(25/26) = 5/√26
cos(x/2) = √[(1+cos x)/2] = √[(1-12/13)/2] = √[(1/13)/2] = √(1/26) = 1/√26
tan(x/2) = sin(x/2)/cos(x/2) = 5

30. Find the conjugate of (√(5+12i) + √(5-12i))/(√(5+12i) - √(5-12i))

Solution:

Let z = (√a + √b)/(√a - √b) where a = 5+12i, b = 5-12i
Rationalize: z = (√a + √b)²/(a-b) = (a+b+2√ab)/(a-b)
a+b = 10, a-b = 24i, √ab = √(25+144) = √169 = 13
z = (10+26)/(24i) = 36/(24i) = 3/(2i) = -3i/2
Conjugate of z = 3i/2

31. The sum of two numbers is 6 times their geometric mean, show that numbers are in the ratio (3+2√2):(3-2√2)

Solution:

Let numbers be a and b, GM = √(ab)
Given: a+b = 6√(ab)
Divide by b: a/b + 1 = 6√(a/b)
Let t = √(a/b): t² + 1 = 6t
t² - 6t + 1 = 0 ⇒ t = (6±√32)/2 = 3±2√2
a/b = t² = (3±2√2)² = 17±12√2
Ratio = (3+2√2):(3-2√2)

SECTION D: Long Answer Type Questions

32. Prove that: (cos x - cos y)² + (sin x - sin y)² = 4sin²((x-y)/2)

Solution:

LHS = cos²x - 2cos x cos y + cos²y + sin²x - 2sin x sin y + sin²y
= (cos²x+sin²x) + (cos²y+sin²y) - 2(cos x cos y + sin x sin y)
= 1 + 1 - 2cos(x-y)
= 2 - 2cos(x-y)
= 2[1 - cos(x-y)]
= 2[2sin²((x-y)/2)]
= 4sin²((x-y)/2) = RHS

33. A person standing at the junction of two straight paths represented by 2x-3y+4=0 and 3x+4y-5=0 wants to reach the path 6x-7y+8=0 in the least time. Find equation of the path that he should follow.

Solution:

Solve: 2x-3y+4=0 and 3x+4y-5=0
Multiply first by 3: 6x-9y+12=0
Multiply second by 2: 6x+8y-10=0
Subtract: -17y+22=0 ⇒ y=22/17
Substitute: 2x-3(22/17)+4=0 ⇒ 2x=66/17-68/17=-2/17 ⇒ x=-1/17
Junction point: (-1/17, 22/17)
Shortest path is perpendicular to target line 6x-7y+8=0
Slope of target line = 6/7, so perpendicular slope = -7/6
Equation: y-22/17 = -7/6(x+1/17)

34. The cable of a uniformly loaded suspension bridge hangs in the form of a parabola. The roadway which is horizontal and 100 m long is supported by vertical wires attached to the cable, the longest wire being 30 m and the shortest being 6 m. Find the length of a supporting wire attached to the roadway 18 m from the middle.

Solution:

Place parabola with vertex at origin, opening upward: x² = 4ay
Roadway from x=-50 to x=50
At ends: 50² = 4a(30) ⇒ 2500 = 120a ⇒ a = 2500/120 = 125/6
At middle: 0² = 4a(6) ⇒ 0 = 24a ⇒ This doesn't work
Better approach: Vertex is at middle, but not at roadway level
Let vertex be (0,k), equation: x² = 4a(y-k)
At x=±50, y=30: 2500 = 4a(30-k)
At x=0, y=6: 0 = 4a(6-k) ⇒ k=6
Then 2500 = 4a(24) ⇒ 100a = 2500 ⇒ a = 25
Equation: x² = 100(y-6)
At x=18: 324 = 100(y-6) ⇒ y-6 = 3.24 ⇒ y = 9.24 m

35. Find the value of (x+1)⁶ + (x-1)⁶, hence or otherwise evaluate (√2+1)⁶ + (√2-1)⁶

Solution:

Using binomial expansion:
(x+1)⁶ = x⁶ + 6x⁵ + 15x⁴ + 20x³ + 15x² + 6x + 1
(x-1)⁶ = x⁶ - 6x⁵ + 15x⁴ - 20x³ + 15x² - 6x + 1
Sum = 2x⁶ + 30x⁴ + 30x² + 2
For x=√2: (√2)⁶=8, (√2)⁴=4, (√2)²=2
Sum = 2×8 + 30×4 + 30×2 + 2 = 16+120+60+2 = 198

SECTION E: Case Study Based Questions

36. Vijay and Vivek are two students of class XI. Their ages are represented by functions f(x) = √(x+4) and g(x) = √(16-x²).

Solution:

(i) Domain of f(x): x+4 ≥ 0 ⇒ x ≥ -4 ⇒ [-4,∞)
Domain of g(x): 16-x² ≥ 0 ⇒ x² ≤ 16 ⇒ -4 ≤ x ≤ 4 ⇒ [-4,4]
(ii) Domain of sum f(x)+g(x): Intersection of domains ⇒ [-4,4]
(iii) Domain of f(g(x)): First g(x) must be defined ⇒ x ∈ [-4,4]
Then f(g(x)) = √(g(x)+4) = √(√(16-x²)+4) defined for all x in [-4,4]

37. Rahul places pots such that number of pots in rows follow pattern: 2, 4, 8, ...

Solution:

This is a GP with a=2, r=2
(i) Pots in 8th row = ar⁷ = 2×2⁷ = 2⁸ = 256
(ii) Total pots in 10 rows = a(r¹⁰-1)/(r-1) = 2(2¹⁰-1)/(2-1) = 2(1024-1) = 2046
(iii) For 510 pots: 2(2ⁿ-1) = 510 ⇒ 2ⁿ-1 = 255 ⇒ 2ⁿ = 256 ⇒ n=8 rows

38. Global Tech Inc. has 10 employees (6 men, 4 women). Committee of 5 members to be formed.

Solution:

(i) At least 2 ladies:
Case 1: 2 ladies, 3 gents: C(4,2)×C(6,3) = 6×20 = 120
Case 2: 3 ladies, 2 gents: C(4,3)×C(6,2) = 4×15 = 60
Case 3: 4 ladies, 1 gent: C(4,4)×C(6,1) = 1×6 = 6

Case 3: 2 ladies, 3 gents: C(4,2)×C(6,3) = 6×20 = 120
Total = 6+60+120 = 186
