Relations and Functions - Concepts & Formulas
Chapter Concepts

A relation R in a set A is called:

  • Reflexive if $(a,a) \in R$ for every $a \in A$
  • Symmetric if $(a,b) \in R \Rightarrow (b,a) \in R$ for every $a,b \in A$
  • Transitive if $(a,b) \in R$ and $(b,c) \in R \Rightarrow (a,c) \in R$ for every $a,b,c \in A$

A relation is an equivalence relation if it is reflexive, symmetric and transitive.

A function $f: X \subset \mathbb{R} \to Y \subset \mathbb{R}$ is:

  • One-one (injective) if the images of distinct elements of X under f are distinct
  • Onto (surjective) if every element of Y is the image of some element of X
  • Bijective if it is both one-one and onto
Key Formulas & Properties
  • Number of relations from A to B: $2^{mn}$ where |A|=m, |B|=n
  • Number of functions from A to B: $n^m$
  • Number of one-one functions from A to B: $^nP_m$ if $n \ge m$, else 0
  • Composition of functions: $(f \circ g)(x) = f(g(x))$
  • Inverse function exists only for bijective functions
Important Types of Functions
  • Identity function: $f(x) = x$
  • Constant function: $f(x) = c$
  • Modulus function: $f(x) = |x|$
  • Signum function: $f(x) = \frac{|x|}{x}$ for $x \ne 0$, $f(0) = 0$
  • Greatest integer function: $f(x) = [x]$

Inverse Trigonometric Functions - Concepts & Formulas
Chapter Concepts

Inverse trigonometric functions are the inverse functions of trigonometric functions. They are defined with restricted domains to make them one-to-one and hence invertible.

Principal Value Branches:

  • $\sin^{-1}x$: Domain $[-1,1]$, Range $[-\frac{\pi}{2}, \frac{\pi}{2}]$
  • $\cos^{-1}x$: Domain $[-1,1]$, Range $[0, \pi]$
  • $\tan^{-1}x$: Domain $\mathbb{R}$, Range $(-\frac{\pi}{2}, \frac{\pi}{2})$
  • $\cot^{-1}x$: Domain $\mathbb{R}$, Range $(0, \pi)$
  • $\sec^{-1}x$: Domain $\mathbb{R}-(-1,1)$, Range $[0, \pi] - \{\frac{\pi}{2}\}$
  • $\cosec^{-1}x$: Domain $\mathbb{R}-(-1,1)$, Range $[-\frac{\pi}{2}, \frac{\pi}{2}] - \{0\}$
Key Formulas & Properties
  • $\sin^{-1}(-x) = -\sin^{-1}x$
  • $\cos^{-1}(-x) = \pi - \cos^{-1}x$
  • $\tan^{-1}(-x) = -\tan^{-1}x$
  • $\sin^{-1}x + \cos^{-1}x = \frac{\pi}{2}$
  • $\tan^{-1}x + \cot^{-1}x = \frac{\pi}{2}$
  • $\sec^{-1}x + \cosec^{-1}x = \frac{\pi}{2}$
  • $\tan^{-1}x + \tan^{-1}y = \tan^{-1}\left(\frac{x+y}{1-xy}\right)$ if $xy < 1$
  • $\tan^{-1}x - \tan^{-1}y = \tan^{-1}\left(\frac{x-y}{1+xy}\right)$ if $xy > -1$
Important Conversions
  • $\sin^{-1}x = \cos^{-1}\sqrt{1-x^2} = \tan^{-1}\frac{x}{\sqrt{1-x^2}}$
  • $\cos^{-1}x = \sin^{-1}\sqrt{1-x^2} = \tan^{-1}\frac{\sqrt{1-x^2}}{x}$
  • $\tan^{-1}x = \sin^{-1}\frac{x}{\sqrt{1+x^2}} = \cos^{-1}\frac{1}{\sqrt{1+x^2}}$
  • $2\tan^{-1}x = \sin^{-1}\frac{2x}{1+x^2} = \cos^{-1}\frac{1-x^2}{1+x^2} = \tan^{-1}\frac{2x}{1-x^2}$

Matrices and Determinants - Concepts & Formulas
Chapter Concepts

A matrix is a rectangular array of numbers arranged in rows and columns. Determinants are scalar values that can be computed from square matrices.

Types of Matrices:

  • Row Matrix: Matrix with only one row
  • Column Matrix: Matrix with only one column
  • Square Matrix: Matrix with equal number of rows and columns
  • Diagonal Matrix: Square matrix with all non-diagonal elements zero
  • Scalar Matrix: Diagonal matrix with all diagonal elements equal
  • Identity Matrix: Diagonal matrix with all diagonal elements 1
  • Zero Matrix: Matrix with all elements zero
Key Formulas & Properties
  • $A(adjA) = (adjA)A = |A|I$
  • $(AB)^{-1} = B^{-1}A^{-1}$
  • $(AB)^T = B^TA^T$
  • $|adjA| = |A|^{n-1}$ where n is the order of matrix A
  • If $|A| \ne 0$, then $A^{-1} = \frac{1}{|A|} adjA$
  • For a square matrix A, A is:
    • Symmetric if $A^T = A$
    • Skew-symmetric if $A^T = -A$
    • Orthogonal if $AA^T = A^TA = I$
Determinant Properties
  • $|AB| = |A||B|$
  • $|A^T| = |A|$
  • $|A^{-1}| = \frac{1}{|A|}$
  • $|kA| = k^n|A|$ where n is order of A
  • If any two rows/columns are identical, then $|A| = 0$
  • If any row/column is all zeros, then $|A| = 0$

Continuity and Differentiability - Concepts & Formulas
Chapter Concepts

A function f is continuous at a point a if $\lim_{x \to a} f(x) = f(a)$. A function is differentiable at a point if its derivative exists at that point.

Continuity:

  • A function is continuous at x=a if:
    1. f(a) exists
    2. $\lim_{x \to a} f(x)$ exists
    3. $\lim_{x \to a} f(x) = f(a)$
  • Every differentiable function is continuous, but converse is not true

Differentiability:

  • Left hand derivative: $f'(a^-) = \lim_{h \to 0^-} \frac{f(a+h)-f(a)}{h}$
  • Right hand derivative: $f'(a^+) = \lim_{h \to 0^+} \frac{f(a+h)-f(a)}{h}$
  • f is differentiable at x=a if $f'(a^-) = f'(a^+)$
Key Formulas
  • $\frac{d}{dx}(x^n) = nx^{n-1}$
  • $\frac{d}{dx}(\sin x) = \cos x$
  • $\frac{d}{dx}(\cos x) = -\sin x$
  • $\frac{d}{dx}(\tan x) = \sec^2 x$
  • $\frac{d}{dx}(\cot x) = -\cosec^2 x$
  • $\frac{d}{dx}(\sec x) = \sec x \tan x$
  • $\frac{d}{dx}(\cosec x) = -\cosec x \cot x$
  • $\frac{d}{dx}(e^x) = e^x$
  • $\frac{d}{dx}(a^x) = a^x \log a$
  • $\frac{d}{dx}(\log x) = \frac{1}{x}$
Inverse Trigonometric Derivatives
  • $\frac{d}{dx}(\sin^{-1} x) = \frac{1}{\sqrt{1-x^2}}$
  • $\frac{d}{dx}(\cos^{-1} x) = -\frac{1}{\sqrt{1-x^2}}$
  • $\frac{d}{dx}(\tan^{-1} x) = \frac{1}{1+x^2}$

Application of Derivatives - Concepts & Formulas
Chapter Concepts

Derivatives have various applications including rate of change, increasing/decreasing functions, and finding maxima/minima.

Rate of Change:

  • $\frac{dy}{dt}$ represents rate of change of y with respect to t
  • If $y = f(x)$ and $x = g(t)$, then $\frac{dy}{dt} = \frac{dy}{dx} \cdot \frac{dx}{dt}$

Increasing/Decreasing Functions:

  • Increasing function: $f'(x) > 0$
  • Decreasing function: $f'(x) < 0$
Maxima and Minima

First Derivative Test:

  • Find critical points where $f'(x) = 0$ or f'(x) does not exist
  • If f'(x) changes from positive to negative at c, then f(c) is local maximum
  • If f'(x) changes from negative to positive at c, then f(c) is local minimum

Second Derivative Test:

  • If $f'(c) = 0$ and $f''(c) < 0$, then f(c) is local maximum
  • If $f'(c) = 0$ and $f''(c) > 0$, then f(c) is local minimum
  • If $f''(c) = 0$, test fails
Practical Applications
  • Optimization problems (maximizing area, volume, profit, etc.)
  • Related rates problems
  • Motion problems
  • Economics applications (marginal cost, revenue, etc.)

Integrals - Concepts & Formulas
Chapter Concepts

Integration is the inverse process of differentiation. It has applications in finding areas, volumes, and solving differential equations.

Indefinite Integrals:

  • $\int f(x) dx = F(x) + C$ where $F'(x) = f(x)$
  • C is the constant of integration

Definite Integrals:

  • $\int_a^b f(x) dx = F(b) - F(a)$ where $F'(x) = f(x)$
  • Definite integral represents the area under the curve
Basic Integration Formulas
  • $\int x^n dx = \frac{x^{n+1}}{n+1} + C$, $n \ne -1$
  • $\int \frac{1}{x} dx = \log|x| + C$
  • $\int e^x dx = e^x + C$
  • $\int a^x dx = \frac{a^x}{\log a} + C$
  • $\int \sin x dx = -\cos x + C$
  • $\int \cos x dx = \sin x + C$
  • $\int \sec^2 x dx = \tan x + C$
  • $\int \cosec^2 x dx = -\cot x + C$
  • $\int \sec x \tan x dx = \sec x + C$
  • $\int \cosec x \cot x dx = -\cosec x + C$
Integration Methods
  • Substitution: $\int f(g(x))g'(x) dx = \int f(u) du$
  • Integration by parts: $\int u v dx = u \int v dx - \int u'(\int v dx) dx$
  • Partial fractions: For rational functions
  • Trigonometric substitutions: For expressions with $\sqrt{a^2 - x^2}$, $\sqrt{a^2 + x^2}$, etc.

Application of Integrals - Concepts & Formulas
Chapter Concepts

Integrals are used to find areas under curves and between curves. The area bounded by the curve y=f(x), x-axis and the ordinates x=a and x=b is given by $\int_a^b f(x) dx$.

Area Formulas:

  • Area under curve: $\int_a^b f(x) dx$
  • Area between two curves: $\int_a^b [f(x) - g(x)] dx$ where $f(x) \ge g(x)$
  • Area bounded by curve x=f(y) and y-axis: $\int_c^d f(y) dy$
  • Area enclosed by ellipse: $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ is $\pi ab$
  • Area enclosed by circle: $x^2 + y^2 = a^2$ is $\pi a^2$
Area Calculation Methods
  • Vertical strips: Area = $\int_a^b [f(x) - g(x)] dx$
  • Horizontal strips: Area = $\int_c^d [f(y) - g(y)] dy$
  • Polar coordinates: Area = $\frac{1}{2} \int_\alpha^\beta r^2 d\theta$
  • Parametric form: Area = $\int_{t_1}^{t_2} y \frac{dx}{dt} dt$
Important Areas
  • Area of triangle using coordinates: $\frac{1}{2} |x_1(y_2-y_3) + x_2(y_3-y_1) + x_3(y_1-y_2)|$
  • Area of sector: $\frac{1}{2} r^2 \theta$
  • Area of segment: $\frac{1}{2} r^2 (\theta - \sin \theta)$

Differential Equations - Concepts & Formulas
Chapter Concepts

A differential equation is an equation involving derivatives of a function. The order is the order of the highest derivative, and the degree is the power of the highest order derivative.

Types of Differential Equations:

  • Variable separable: $\frac{dy}{dx} = f(x)g(y)$
  • Homogeneous: $\frac{dy}{dx} = f\left(\frac{y}{x}\right)$
  • Linear: $\frac{dy}{dx} + P(x)y = Q(x)$
  • Exact: $M dx + N dy = 0$ where $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$
Solution Methods
  • Variable separable: $\int \frac{dy}{g(y)} = \int f(x) dx$
  • Homogeneous: Substitute $y = vx$
  • Linear: Integrating factor $e^{\int P dx}$
  • Exact: Find $F(x,y)$ such that $dF = M dx + N dy$
Important Differential Equations
  • Exponential growth/decay: $\frac{dy}{dt} = ky$
  • Newton's law of cooling: $\frac{dT}{dt} = -k(T - T_0)$
  • Population growth: $\frac{dP}{dt} = kP\left(1 - \frac{P}{M}\right)$
  • Simple harmonic motion: $\frac{d^2x}{dt^2} + \omega^2 x = 0$

Vectors - Concepts & Formulas
Chapter Concepts

Vectors have both magnitude and direction. They can be represented in component form or using unit vectors i, j, k.

Vector Operations:

  • Addition: Triangle law, parallelogram law
  • Subtraction: $\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$
  • Scalar multiplication: $k\vec{a}$ has magnitude $|k||\vec{a}|$
  • Dot product: $\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$
  • Cross product: $\vec{a} \times \vec{b} = |\vec{a}||\vec{b}|\sin\theta \hat{n}$
Key Formulas & Properties
  • Dot product: $\vec{a} \cdot \vec{b} = a_xb_x + a_yb_y + a_zb_z$
  • Cross product: $\vec{a} \times \vec{b} = \begin{vmatrix} \hat{i} & \hat{j} & \hat{k} \\ a_x & a_y & a_z \\ b_x & b_y & b_z \end{vmatrix}$
  • Scalar projection of $\vec{a}$ on $\vec{b}$: $\frac{\vec{a} \cdot \vec{b}}{|\vec{b}|}$
  • Vector projection of $\vec{a}$ on $\vec{b}$: $\left(\frac{\vec{a} \cdot \vec{b}}{|\vec{b}|^2}\right) \vec{b}$
  • Area of parallelogram = $|\vec{a} \times \vec{b}|$
  • Area of triangle = $\frac{1}{2} |\vec{a} \times \vec{b}|$

Three-Dimensional Geometry - Concepts & Formulas
Chapter Concepts

3D geometry deals with points, lines, and planes in three-dimensional space. Direction cosines and ratios are used to represent the direction of lines.

Direction Cosines and Ratios:

  • Direction cosines: $l = \cos\alpha$, $m = \cos\beta$, $n = \cos\gamma$
  • $l^2 + m^2 + n^2 = 1$
  • Direction ratios: $a, b, c$ proportional to $l, m, n$

Line Equations:

  • Vector form: $\vec{r} = \vec{a} + \lambda\vec{b}$
  • Cartesian form: $\frac{x-x_1}{a} = \frac{y-y_1}{b} = \frac{z-z_1}{c}$
Key Formulas
  • Distance between points: $\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$
  • Angle between lines: $\cos\theta = \frac{|a_1a_2 + b_1b_2 + c_1c_2|}{\sqrt{a_1^2+b_1^2+c_1^2}\sqrt{a_2^2+b_2^2+c_2^2}}$
  • Shortest distance between skew lines: $\frac{|(\vec{b_1} \times \vec{b_2}) \cdot (\vec{a_2} - \vec{a_1})|}{|\vec{b_1} \times \vec{b_2}|}$
  • Distance from point to line: $\frac{|\vec{AB} \times \vec{b}|}{|\vec{b}|}$
  • Distance between parallel lines: $\frac{|\vec{AB} \times \vec{b}|}{|\vec{b}|}$

Linear Programming - Concepts & Formulas
Chapter Concepts

Linear programming involves optimizing a linear objective function subject to linear constraints. The graphical method is used for problems with two variables.

Key Terms:

  • Objective function: Function to be maximized/minimized
  • Constraints: Linear inequalities representing restrictions
  • Feasible region: Region satisfying all constraints
  • Optimal solution: Point in feasible region optimizing objective function
  • Corner point method: Evaluate objective function at all corner points
Solution Methods
  • Graphical method: For problems with 2 variables
    1. Plot constraints to find feasible region
    2. Find corner points of feasible region
    3. Evaluate objective function at each corner point
    4. Optimal solution is at a corner point
  • Simplex method: For problems with more variables
Important Results
  • Feasible region is always a convex polygon
  • Optimal solution always occurs at a corner point
  • If feasible region is unbounded, optimal solution may not exist
  • If two corner points give same optimal value, all points on the line segment between them are optimal

Probability - Concepts & Formulas
Chapter Concepts

Probability measures the likelihood of events. Conditional probability, Bayes' theorem, and probability distributions are important concepts.

Basic Concepts:

  • Sample space: Set of all possible outcomes
  • Event: Subset of sample space
  • Probability: $P(E) = \frac{\text{Number of favorable outcomes}}{\text{Total number of outcomes}}$
  • Complement: $P(E') = 1 - P(E)$
  • Union: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
  • Intersection: $P(A \cap B) = P(A)P(B|A) = P(B)P(A|B)$
Key Formulas
  • Conditional probability: $P(A|B) = \frac{P(A \cap B)}{P(B)}$
  • Multiplication theorem: $P(A \cap B) = P(A)P(B|A) = P(B)P(A|B)$
  • Bayes' theorem: $P(E_i|A) = \frac{P(E_i)P(A|E_i)}{\sum P(E_j)P(A|E_j)}$
  • Mean of random variable: $E(X) = \sum x_iP(x_i)$
  • Variance: $Var(X) = E(X^2) - [E(X)]^2$
  • Standard deviation: $\sigma = \sqrt{Var(X)}$
