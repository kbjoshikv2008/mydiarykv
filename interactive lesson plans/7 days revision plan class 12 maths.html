Day 1: Relations & Functions

6 Questions
Question 1
1 Mark

A function f : A → B defined as \( f(x) = 2x \) is both one-one and onto. If A = {1, 2, 3, 4}, then find the set B.

Solution:

Since f is one-one and onto, it is a bijection.

f(1) = 2, f(2) = 4, f(3) = 6, f(4) = 8

Therefore, B = {2, 4, 6, 8}

Question 2
2 Marks

Check whether the relation S in the set of all real numbers (\(\mathbb{R}\)) defined by \( S = \{(a, b) : a \leq b^3\} \) is reflexive, symmetric or transitive.

Solution:

Reflexive: For any a ∈ R, a ≤ a³ is not always true (e.g., a=2, 2 ≤ 8 is true; a=1/2, 1/2 ≤ 1/8 is false). So not reflexive.

Symmetric: If a ≤ b³, it doesn't imply b ≤ a³. Counterexample: a=1, b=2. 1 ≤ 8 is true but 2 ≤ 1 is false. So not symmetric.

Transitive: If a ≤ b³ and b ≤ c³, it doesn't necessarily imply a ≤ c³. Counterexample: a=1/2, b=1, c=2. 1/2 ≤ 1 is true, 1 ≤ 8 is true, but 1/2 ≤ 8 is false. So not transitive.

Therefore, S is neither reflexive, nor symmetric, nor transitive.

Question 3
3 Marks

A function f: \([-4, 4] \rightarrow [0, 4]\) is given by \(f(x) = \sqrt{16 - x^2}\). Show that f is an onto function but not a one-one function. Further, find all possible values of 'a' for which \(f(a) = \sqrt{7}\).

Solution:

f(x) = \(\sqrt{16 - x^2}\)

Onto: For any y ∈ [0,4], we can find x = ±√(16-y²) ∈ [-4,4] such that f(x) = y. So f is onto.

Not one-one: f(x) = f(-x) for all x ∈ [-4,4] but x ≠ -x (except 0). So f is not one-one.

For f(a) = √7: √(16-a²) = √7 ⇒ 16-a² = 7 ⇒ a² = 9 ⇒ a = ±3

So the possible values of a are 3 and -3.

Question 4
2 Marks

Let A = {3, 5}. Then number of reflexive relations on A is

(a) 2 (b) 4 (c) 0 (d) 8

Solution:

For a set with n elements, number of reflexive relations = \(2^{n^2 - n}\)

Here n = 2, so number of reflexive relations = \(2^{4-2} = 2^2 = 4\)

Therefore, the correct answer is (b) 4.

Question 5
4 Marks

Let R be the relation defined in the set A = {1, 2, 3, 4, 5, 6, 7} by R = { (a, b) : both a and b are either odd or even}. Show that R is an equivalence relation. Hence, find the elements of equivalence class [1].

Solution:

Reflexive: For any a ∈ A, a and a have same parity (both odd or both even). So (a,a) ∈ R. Hence reflexive.

Symmetric: If (a,b) ∈ R, then a and b have same parity, so b and a have same parity. Hence (b,a) ∈ R. So symmetric.

Transitive: If (a,b) ∈ R and (b,c) ∈ R, then a and b have same parity, and b and c have same parity. So a and c have same parity. Hence (a,c) ∈ R. So transitive.

Therefore, R is an equivalence relation.

Equivalence class [1] = {x ∈ A : (1,x) ∈ R} = {all odd numbers in A} = {1, 3, 5, 7}

Question 6
4 Marks

A function f : \( R_+ \to R \) (where \( R_+ \) is the set of all non-negative real numbers) defined by \( f(x) = 4x + 3 \) is :

(A) one-one but not onto

(B) onto but not one-one

(C) both one-one and onto

(D) neither one-one nor onto

Solution:

One-one: Let f(x₁) = f(x₂) ⇒ 4x₁+3 = 4x₂+3 ⇒ x₁ = x₂. So f is one-one.

Onto: Let y ∈ R (codomain). Then f(x) = y ⇒ 4x+3 = y ⇒ x = (y-3)/4

For y < 3, x becomes negative, but domain is R₊ (non-negative reals). So f is not onto.

Therefore, f is one-one but not onto.

The correct answer is (A) one-one but not onto.

Day 2: Inverse Trigonometric Functions

6 Questions
Question 1
2 Marks

Evaluate: \(\sin^{-1} \left( \sin \frac{3\pi}{4} \right) + \cos^{-1} \left( \cos \frac{3\pi}{4} \right) + \tan^{-1}(1)\)

Solution:

\(\sin^{-1} \left( \sin \frac{3\pi}{4} \right) = \sin^{-1} \left( \sin \left( \pi - \frac{\pi}{4} \right) \right) = \sin^{-1} \left( \sin \frac{\pi}{4} \right) = \frac{\pi}{4}\)

\(\cos^{-1} \left( \cos \frac{3\pi}{4} \right) = \cos^{-1} \left( \cos \left( \pi - \frac{\pi}{4} \right) \right) = \cos^{-1} \left( -\cos \frac{\pi}{4} \right) = \pi - \frac{\pi}{4} = \frac{3\pi}{4}\)

\(\tan^{-1}(1) = \frac{\pi}{4}\)

Sum = \(\frac{\pi}{4} + \frac{3\pi}{4} + \frac{\pi}{4} = \frac{5\pi}{4}\)

Question 2
3 Marks

Evaluate: \(3 \sin^{-1} \left( \frac{1}{\sqrt{2}} \right) + 2 \cos^{-1} \left( \frac{\sqrt{3}}{2} \right) + \cos^{-1}(0)\)

Solution:

\(\sin^{-1} \left( \frac{1}{\sqrt{2}} \right) = \frac{\pi}{4}\)

\(\cos^{-1} \left( \frac{\sqrt{3}}{2} \right) = \frac{\pi}{6}\)

\(\cos^{-1}(0) = \frac{\pi}{2}\)

Sum = \(3 \times \frac{\pi}{4} + 2 \times \frac{\pi}{6} + \frac{\pi}{2} = \frac{3\pi}{4} + \frac{\pi}{3} + \frac{\pi}{2}\)

= \(\frac{9\pi + 4\pi + 6\pi}{12} = \frac{19\pi}{12}\)

Question 3
2 Marks

Find the domain of \( y = \sin^{-1} (x^2 - 4) \).

Solution:

For \(\sin^{-1}(x^2 - 4)\) to be defined:

\(-1 \leq x^2 - 4 \leq 1\)

\(\Rightarrow 3 \leq x^2 \leq 5\)

\(\Rightarrow x \in [-\sqrt{5}, -\sqrt{3}] \cup [\sqrt{3}, \sqrt{5}]\)

Therefore, domain = \([-\sqrt{5}, -\sqrt{3}] \cup [\sqrt{3}, \sqrt{5}]\)

Question 4
3 Marks

Evaluate: \(\cos^{-1} \left[ \cos \left( -\frac{7\pi}{3} \right) \right]\)

Solution:

\(\cos^{-1} \left[ \cos \left( -\frac{7\pi}{3} \right) \right] = \cos^{-1} \left[ \cos \left( -2\pi - \frac{\pi}{3} \right) \right]\)

= \(\cos^{-1} \left[ \cos \left( 2\pi + \frac{\pi}{3} \right) \right]\) (since cosine is even)

= \(\cos^{-1} \left[ \cos \frac{\pi}{3} \right]\) (since cosine has period \(2\pi\))

= \(\frac{\pi}{3}\) (since \(\frac{\pi}{3} \in [0, \pi]\))

Question 5
4 Marks

Find the value of \(\cos^{-1} \left( \frac{1}{2} \right) - \tan^{-1} \left( -\frac{1}{\sqrt{3}} \right) + \csc^{-1} (-2)\)

Solution:

\(\cos^{-1} \left( \frac{1}{2} \right) = \frac{\pi}{3}\)

\(\tan^{-1} \left( -\frac{1}{\sqrt{3}} \right) = -\frac{\pi}{6}\)

\(\csc^{-1} (-2) = -\frac{\pi}{6}\) (since \(\csc(-\frac{\pi}{6}) = -2\))

Sum = \(\frac{\pi}{3} - \left( -\frac{\pi}{6} \right) + \left( -\frac{\pi}{6} \right) = \frac{\pi}{3} + \frac{\pi}{6} - \frac{\pi}{6} = \frac{\pi}{3}\)

Question 6
4 Marks

Simplify: \(\tan^{-1} \left( \frac{\cos x}{1 - \sin x} \right)\), where \(\frac{-\pi}{2} < x < \frac{\pi}{2}\)

Solution:

\(\tan^{-1} \left( \frac{\cos x}{1 - \sin x} \right)\)

= \(\tan^{-1} \left( \frac{\cos^2(x/2) - \sin^2(x/2)}{\cos^2(x/2) + \sin^2(x/2) - 2\sin(x/2)\cos(x/2)} \right)\)

= \(\tan^{-1} \left( \frac{(\cos(x/2) - \sin(x/2))(\cos(x/2) + \sin(x/2))}{(\cos(x/2) - \sin(x/2))^2} \right)\)

= \(\tan^{-1} \left( \frac{\cos(x/2) + \sin(x/2)}{\cos(x/2) - \sin(x/2)} \right)\)

= \(\tan^{-1} \left( \frac{1 + \tan(x/2)}{1 - \tan(x/2)} \right)\)

= \(\tan^{-1} \left( \tan\left(\frac{\pi}{4} + \frac{x}{2}\right) \right)\)

= \(\frac{\pi}{4} + \frac{x}{2}\) (since \(\frac{-\pi}{2} < x < \frac{\pi}{2} \Rightarrow \frac{-\pi}{4} < \frac{x}{2} < \frac{\pi}{4}\))

Question 1
1 Mark

If \( A = \begin{bmatrix} 0 & 1 \\ -1 & 0 \end{bmatrix} \) and \((3I + 4A)(3I - 4A) = x^2I\), then the value(s) x is/are :

(a) \(\pm \sqrt{7}\)

(b) 0

(c) \(\pm 5\)

(d) 25

Solution:

First, calculate \(3I + 4A\) and \(3I - 4A\):

\(3I = \begin{bmatrix} 3 & 0 \\ 0 & 3 \end{bmatrix}\), \(4A = \begin{bmatrix} 0 & 4 \\ -4 & 0 \end{bmatrix}\)

\(3I + 4A = \begin{bmatrix} 3 & 4 \\ -4 & 3 \end{bmatrix}\), \(3I - 4A = \begin{bmatrix} 3 & -4 \\ 4 & 3 \end{bmatrix}\)

Now multiply: \((3I + 4A)(3I - 4A) = \begin{bmatrix} 3 & 4 \\ -4 & 3 \end{bmatrix} \begin{bmatrix} 3 & -4 \\ 4 & 3 \end{bmatrix}\)

= \(\begin{bmatrix} 9+16 & -12+12 \\ -12+12 & 16+9 \end{bmatrix} = \begin{bmatrix} 25 & 0 \\ 0 & 25 \end{bmatrix} = 25I\)

So \(x^2 = 25 \Rightarrow x = \pm 5\)

Answer: (c) \(\pm 5\)

Question 2
1 Mark

If for a square matrix \( A, A^2 - 3A + I = 0 \) and \( A^{-1} = xA + yI \), then the value of \( x + y \) is :

(a) \(-2\)

(b) 2

(c) 3

(d) \(-3\)

Solution:

Given: \(A^2 - 3A + I = 0\)

Multiply both sides by \(A^{-1}\): \(A - 3I + A^{-1} = 0\)

So \(A^{-1} = 3I - A\)

Comparing with \(A^{-1} = xA + yI\), we get \(x = -1, y = 3\)

Thus \(x + y = -1 + 3 = 2\)

Answer: (b) 2

Question 4
3 Marks

If \( A \) is a \( 2 \times 3 \) matrix such that \( AB \) and \( AB' \) both are defined, then order of the matrix \( B \) is

(A) \( 2 \times 2 \)

(B) \( 2 \times 1 \)

(C) \( 3 \times 2 \)

(D) \( 3 \times 3 \)

Solution:

For AB to be defined, number of columns of A must equal number of rows of B.

A is 2×3, so B must have 3 rows.

For AB' to be defined, number of columns of A must equal number of rows of B'.

B' has same number of rows as B has columns.

So if B is m×n, then AB is defined ⇒ n = 3

AB' is defined ⇒ m = 3

So B is 3×3

Answer: (D) \( 3 \times 3 \)

Question 5
4 Marks

If \(\begin{bmatrix} 2 & 0 \\ 5 & 4 \end{bmatrix} = P + Q\), where \( P \) is a symmetric and \( Q \) is a skew symmetric matrix, then \( Q \) is equal to

(A) \(\begin{bmatrix} 2 & 5/2 \\ 5/2 & 4 \end{bmatrix}\)

(B) \(\begin{bmatrix} 0 & -5/2 \\ 5/2 & 0 \end{bmatrix}\)

(C) \(\begin{bmatrix} 0 & 5/2 \\ -5/2 & 0 \end{bmatrix}\)

(D) \(\begin{bmatrix} 2 & -5/2 \\ 5/2 & 4 \end{bmatrix}\)

Solution:

Any square matrix can be expressed as sum of symmetric and skew-symmetric matrices:

\(P = \frac{1}{2}(A + A^T)\), \(Q = \frac{1}{2}(A - A^T)\)

Here \(A = \begin{bmatrix} 2 & 0 \\ 5 & 4 \end{bmatrix}\), \(A^T = \begin{bmatrix} 2 & 5 \\ 0 & 4 \end{bmatrix}\)

\(A - A^T = \begin{bmatrix} 0 & -5 \\ 5 & 0 \end{bmatrix}\)

So \(Q = \frac{1}{2} \begin{bmatrix} 0 & -5 \\ 5 & 0 \end{bmatrix} = \begin{bmatrix} 0 & -5/2 \\ 5/2 & 0 \end{bmatrix}\)

Answer: (B) \(\begin{bmatrix} 0 & -5/2 \\ 5/2 & 0 \end{bmatrix}\)

Question 6
4 Marks

If \(\begin{bmatrix} 1 & 2 & 1 \\ 2 & 3 & 1 \\ 3 & a & 1 \end{bmatrix}\) is non-singular matrix and \(a \in A\), then the set \(A\) is

(A) \(\mathbb{R}\)

(B) \(\{0\}\)

(C) \(\{4\}\)

(D) \(\mathbb{R} - \{4\}\)

Solution:

A matrix is non-singular if its determinant ≠ 0

Let \(M = \begin{bmatrix} 1 & 2 & 1 \\ 2 & 3 & 1 \\ 3 & a & 1 \end{bmatrix}\)

\(|M| = 1(3-1) - 2(2-3) + 1(2a-9) = 2 + 2 + 2a - 9 = 2a - 5\)

For non-singular matrix: \(2a - 5 \neq 0 \Rightarrow a \neq \frac{5}{2}\)

But wait, let's recalculate:

Using cofactor expansion: \(|M| = 1(3×1 - 1×a) - 2(2×1 - 1×3) + 1(2×a - 3×3)\)

= \(1(3-a) - 2(2-3) + 1(2a-9) = 3-a +2 + 2a-9 = a - 4\)

So \(|M| = a - 4\)

For non-singular: \(a - 4 \neq 0 \Rightarrow a \neq 4\)

Thus \(A = \mathbb{R} - \{4\}\)

Answer: (D) \(\mathbb{R} - \{4\}\)

If \(|A| = 2\), where \(A\) is a \(2 \times 2\) matrix, then \(|4A^{-1}|\) equals:

(a) 4

(b) 2

(c) 8

(d) \(\frac{1}{32}\)

Solution:

We know: \(|kA| = k^n |A|\) for n×n matrix

Also: \(|A^{-1}| = \frac{1}{|A|}\)

So \(|4A^{-1}| = 4^2 |A^{-1}| = 16 \times \frac{1}{|A|} = 16 \times \frac{1}{2} = 8\)

Answer: (c) 8

Question 4
3 Marks

Solve the following system of equations by matrix method:

\(x + 2y + 3z = 6\)

\(2x - y + z = 2\)

\(3x + 2y - 2z = 3\)

Solution:

The system can be written as \(AX = B\), where

\(A = \begin{bmatrix} 1 & 2 & 3 \\ 2 & -1 & 1 \\ 3 & 2 & -2 \end{bmatrix}\), \(X = \begin{bmatrix} x \\ y \\ z \end{bmatrix}\), \(B = \begin{bmatrix} 6 \\ 2 \\ 3 \end{bmatrix}\)

\(|A| = 1(2-2) - 2(-4-3) + 3(4+3) = 0 + 14 + 21 = 35\)

Since \(|A| \neq 0\), unique solution exists.

\(A^{-1} = \frac{1}{|A|} adj A\)

adj A = \(\begin{bmatrix} 0 & 10 & 5 \\ 7 & -11 & 5 \\ 7 & 4 & -5 \end{bmatrix}\)

So \(A^{-1} = \frac{1}{35} \begin{bmatrix} 0 & 10 & 5 \\ 7 & -11 & 5 \\ 7 & 4 & -5 \end{bmatrix}\)

\(X = A^{-1}B = \frac{1}{35} \begin{bmatrix} 0 & 10 & 5 \\ 7 & -11 & 5 \\ 7 & 4 & -5 \end{bmatrix} \begin{bmatrix} 6 \\ 2 \\ 3 \end{bmatrix}\)

= \(\frac{1}{35} \begin{bmatrix} 0+20+15 \\ 42-22+15 \\ 42+8-15 \end{bmatrix} = \frac{1}{35} \begin{bmatrix} 35 \\ 35 \\ 35 \end{bmatrix} = \begin{bmatrix} 1 \\ 1 \\ 1 \end{bmatrix}\)

So \(x = 1, y = 1, z = 1\)

Question 5
4 Marks

If (a, b), (c, d) and (e, f) are the vertices of \(\triangle ABC\) and \(\Delta\) denotes the area of \(\triangle ABC\), then

\(\begin{vmatrix} a & c & e \\ b & d & f \\ 1 & 1 & 1 \end{vmatrix}^2\) is equal to

(A) \(2\Delta^2\)

(B) \(4\Delta^2\)

(C) \(2\Delta\)

(D) \(4\Delta\)

Solution:

Area of triangle with vertices (a,b), (c,d), (e,f) is:

\(\Delta = \frac{1}{2} \begin{vmatrix} a & b & 1 \\ c & d & 1 \\ e & f & 1 \end{vmatrix}\)

The given determinant is \(\begin{vmatrix} a & c & e \\ b & d & f \\ 1 & 1 & 1 \end{vmatrix}\)

This is the transpose of the area determinant, and determinant of transpose equals original determinant.

So \(\begin{vmatrix} a & c & e \\ b & d & f \\ 1 & 1 & 1 \end{vmatrix} = \begin{vmatrix} a & b & 1 \\ c & d & 1 \\ e & f & 1 \end{vmatrix} = 2\Delta\)

Thus \(\begin{vmatrix} a & c & e \\ b & d & f \\ 1 & 1 & 1 \end{vmatrix}^2 = (2\Delta)^2 = 4\Delta^2\)

Answer: (B) \(4\Delta^2\)

If \(|A| = |kA|\), where \(A\) is a square matrix of order 2, then sum of all possible values of k is

(A) 1

Day 5: Continuity & Differentiability

6 Questions
Question 1
1 Mark

Examine the continuity of the function \(f(x) = \begin{cases} \frac{1-\cos x}{x^2}, & \text{if } x \ne 0 \\ \frac{1}{2}, & \text{if } x = 0 \end{cases}\) at \(x = 0\).

Solution:

\(\lim_{x \to 0} f(x) = \lim_{x \to 0} \frac{1-\cos x}{x^2} = \lim_{x \to 0} \frac{2\sin^2(x/2)}{x^2} = \lim_{x \to 0} \frac{2(x/2)^2}{x^2} = \frac{1}{2}\)

Also \(f(0) = \frac{1}{2}\)

Since \(\lim_{x \to 0} f(x) = f(0)\), the function is continuous at \(x = 0\).

Question 2
1 Mark

If \(f(x) = \begin{cases} 3x - 2, & 0 < x \leq 1 \\ 2x^2 + ax, & 1 < x < 2 \end{cases}\) is continuous for \(x \in (0, 2)\), then a is equal to:

(A) -4 (B) -7/2 (C) -2 (D) -1

Solution:

For continuity at x = 1:

\(\lim_{x \to 1^-} f(x) = 3(1) - 2 = 1\)

\(\lim_{x \to 1^+} f(x) = 2(1)^2 + a(1) = 2 + a\)

\(f(1) = 3(1) - 2 = 1\)

For continuity, \(2 + a = 1 \Rightarrow a = -1\)

Therefore, the correct answer is (D) -1.

If \(y = \log(\log x)\), find \(\frac{dy}{dx}\).

Solution:

\(y = \log(\log x)\)

\(\frac{dy}{dx} = \frac{1}{\log x} \cdot \frac{1}{x} = \frac{1}{x \log x}\)

If \(x = a(\theta - \sin\theta)\) and \(y = a(1 - \cos\theta)\), find \(\frac{dy}{dx}\).

Solution:

\(\frac{dx}{d\theta} = a(1 - \cos\theta)\), \(\frac{dy}{d\theta} = a(\sin\theta)\)

= \(\frac{2\sin(\theta/2)\cos(\theta/2)}{2\sin^2(\theta/2)} = \cot(\theta/2)\)

Question 5
4 Marks

Differentiate \(2^{\cos^2 x}\) with respect to \(\cos^2 x\).

Solution:

Let \(u = 2^{\cos^2 x} \Rightarrow \frac{du}{dx} = 2^{\cos^2 x} (-2\cos x \sin x) \log 2\)

Let \(v = \cos^2 x \Rightarrow \frac{dv}{dx} = -2\cos x \sin x\)

Now \(\frac{du}{dv} = \left( \frac{du}{dx} \right) / \left( \frac{dv}{dx} \right) = 2^{\cos^2 x} \log 2\)

Question 6
4 Marks

If \(x\sqrt{1+y} + y\sqrt{1+x} = 0, -1 < x < 1, x \ne y\), then prove that \(\frac{dy}{dx} = \frac{-1}{(1+x)^2}\).

Solution:

\(x\sqrt{1+y} + y\sqrt{1+x} = 0\)

\(\Rightarrow x\sqrt{1+y} = -y\sqrt{1+x}\)

\(\Rightarrow x^2(1+y) = y^2(1+x)\)

\(\Rightarrow (x - y)(x + y) + xy(x - y) = 0\)

\(\Rightarrow (x - y)(x + y + xy) = 0\)

Since \(x \ne y\), we have \(x + y + xy = 0\)

\(\Rightarrow y = \frac{-x}{1+x}\)

Differentiating both sides with respect to x:

\(\frac{dy}{dx} = \frac{-(1+x) - (-x)(1)}{(1+x)^2} = \frac{-1}{(1+x)^2}\)

Day 6: Applications of Derivatives

Question 1
1 Mark

If \( f(x) = a(x - \cos x) \) is strictly decreasing in \( R \), then 'a' belongs to:

(a) \(\{0\}\) (b) \((0, \infty)\) (c) \((- \infty, 0)\) (d) \((- \infty, \infty)\)

Solution:

\( f'(x) = a(1 + \sin x) \)

For strictly decreasing: \( f'(x) < 0 \) for all \( x \in R \)

Since \( 1 + \sin x \geq 0 \), for \( f'(x) < 0 \), we need \( a < 0 \)

Therefore, \( a \in (-\infty, 0) \)

Answer: (c) \((- \infty, 0)\)

Question 2
1 Mark

The function \( f(x) = x | x | \), \( x \in R \) is differentiable:

(a) only at \( x = 0 \)

(b) only at \( x = 1 \)

(c) in \( R \)

(d) in \( R - \{0\} \)

Solution:

\( f(x) = \begin{cases} x^2, & x \geq 0 \\ -x^2, & x < 0 \end{cases} \)

\( f'(x) = \begin{cases} 2x, & x \geq 0 \\ -2x, & x < 0 \end{cases} \)

At \( x = 0 \):

Left derivative = \( \lim_{h \to 0^-} \frac{f(0+h)-f(0)}{h} = \lim_{h \to 0^-} \frac{-h^2}{h} = 0 \)

Right derivative = \( \lim_{h \to 0^+} \frac{f(0+h)-f(0)}{h} = \lim_{h \to 0^+} \frac{h^2}{h} = 0 \)

Since both derivatives exist and are equal, f is differentiable at x=0

Therefore, f is differentiable in R

Answer: (c) in \( R \)

Question 3
2 Marks

The median of an equilateral triangle is increasing at the rate of \( 2\sqrt{3} \) cm/s. Find the rate at which its side is increasing.

Solution:

For an equilateral triangle of side a:

Median = \( \frac{\sqrt{3}}{2}a \)

Let m = median, then \( m = \frac{\sqrt{3}}{2}a \)

Differentiating with respect to time t:

\( \frac{dm}{dt} = \frac{\sqrt{3}}{2} \frac{da}{dt} \)

Given \( \frac{dm}{dt} = 2\sqrt{3} \) cm/s

So \( 2\sqrt{3} = \frac{\sqrt{3}}{2} \frac{da}{dt} \)

\( \frac{da}{dt} = 4 \) cm/s

Therefore, the side is increasing at 4 cm/s

Sum of two numbers is 5. If the sum of the cubes of these numbers is least, then find the sum of the squares of these numbers.

Solution:

Let the numbers be x and 5-x

Sum of cubes: \( S = x^3 + (5-x)^3 = x^3 + 125 - 75x + 15x^2 - x^3 = 15x^2 - 75x + 125 \)

\( \frac{dS}{dx} = 30x - 75 \)

For minimum, \( \frac{dS}{dx} = 0 \Rightarrow 30x - 75 = 0 \Rightarrow x = 2.5 \)

\( \frac{d^2S}{dx^2} = 30 > 0 \), so minimum at x = 2.5

Numbers are 2.5 and 2.5

Sum of squares = \( (2.5)^2 + (2.5)^2 = 6.25 + 6.25 = 12.5 \)

Question 5
4 Marks

The perimeter of a rectangular metallic sheet is 300 cm. It is rolled along one of its sides to form a cylinder. Find the dimensions of the rectangular sheet so that volume of cylinder so formed is maximum.

Solution:

Let length = l, breadth = b

Perimeter = 2(l + b) = 300 ⇒ l + b = 150 ⇒ b = 150 - l

When rolled along length, circumference of cylinder = l, height = b

Radius r = \( \frac{l}{2\pi} \), height h = b = 150 - l

Volume V = \( \pi r^2 h = \pi \left( \frac{l}{2\pi} \right)^2 (150 - l) = \frac{l^2(150 - l)}{4\pi} \)

\( \frac{dV}{dl} = \frac{1}{4\pi} [2l(150 - l) - l^2] = \frac{1}{4\pi} [300l - 2l^2 - l^2] = \frac{1}{4\pi} [300l - 3l^2] \)

For maximum volume, \( \frac{dV}{dl} = 0 \Rightarrow 300l - 3l^2 = 0 \Rightarrow l(300 - 3l) = 0 \)

So l = 0 or l = 100

l = 0 is not possible, so l = 100 cm

b = 150 - 100 = 50 cm

\( \frac{d^2V}{dl^2} = \frac{1}{4\pi} [300 - 6l] \)

At l = 100, \( \frac{d^2V}{dl^2} = \frac{1}{4\pi} [300 - 600] = \frac{-300}{4\pi} < 0 \), so maximum

Therefore, dimensions are 100 cm × 50 cm

Question 6
4 Marks

Show that the function \( f(x) = \frac{16 \sin x}{4 + \cos x} - x \), is strictly decreasing in \(\left( \frac{\pi}{2}, \pi \right)\).

Solution:

\( f(x) = \frac{16 \sin x}{4 + \cos x} - x \)

\( f'(x) = \frac{16 \cos x (4 + \cos x) - 16 \sin x (-\sin x)}{(4 + \cos x)^2} - 1 \)

= \( \frac{64 \cos x + 16 \cos^2 x + 16 \sin^2 x}{(4 + \cos x)^2} - 1 \)

= \( \frac{64 \cos x + 16}{(4 + \cos x)^2} - 1 \)

For \( x \in \left( \frac{\pi}{2}, \pi \right) \), cos x < 0

So \( 64 \cos x + 16 < 16 \) and \( (4 + \cos x)^2 > 0 \)

Thus \( \frac{64 \cos x + 16}{(4 + \cos x)^2} < \frac{16}{(4 + \cos x)^2} \)

Since 4 + cos x > 3 (as cos x > -1), we have \( \frac{16}{(4 + \cos x)^2} < \frac{16}{9} < 2 \)

So \( \frac{64 \cos x + 16}{(4 + \cos x)^2} - 1 < 1 \), but more importantly, since cos x is negative, 64 cos x + 16 could be negative

Let's check at x = π: cos π = -1, so numerator = 64(-1) + 16 = -48, so f'(x) = -48/9 - 1 < 0

Therefore, f'(x) < 0 for all \( x \in \left( \frac{\pi}{2}, \pi \right) \)

Hence, f(x) is strictly decreasing in \( \left( \frac{\pi}{2}, \pi \right) \)

Question 3
2 Marks

Evaluate: \(\int \frac{x^3 + x + 1}{x^2 - 1} dx\)

Solution:

First, perform polynomial division:

\(\frac{x^3 + x + 1}{x^2 - 1} = x + \frac{2x + 1}{x^2 - 1}\)

Now, \(\frac{2x + 1}{x^2 - 1} = \frac{2x + 1}{(x-1)(x+1)} = \frac{A}{x-1} + \frac{B}{x+1}\)

Solving: \(A = \frac{3}{2}, B = \frac{1}{2}\)

So the integral becomes:

\(\int x dx + \frac{3}{2} \int \frac{dx}{x-1} + \frac{1}{2} \int \frac{dx}{x+1}\)

= \(\frac{x^2}{2} + \frac{3}{2} \log|x-1| + \frac{1}{2} \log|x+1| + C\)

= \(\frac{x^2}{2} + \frac{1}{2} \log|(x-1)^3(x+1)| + C\)

Question 4
3 Marks

Evaluate: \(\int_0^{\pi/4} \log(1 + \tan x) dx\)

Solution:

Let \(I = \int_0^{\pi/4} \log(1 + \tan x) dx\)

Using property: \(\int_0^a f(x) dx = \int_0^a f(a-x) dx\)

\(I = \int_0^{\pi/4} \log(1 + \tan(\pi/4 - x)) dx\)

= \(\int_0^{\pi/4} \log(1 + \frac{1 - \tan x}{1 + \tan x}) dx\)

= \(\int_0^{\pi/4} \log(\frac{2}{1 + \tan x}) dx\)

= \(\int_0^{\pi/4} [\log 2 - \log(1 + \tan x)] dx\)

= \(\int_0^{\pi/4} \log 2 dx - I\)

So, \(2I = \frac{\pi}{4} \log 2\)

Therefore, \(I = \frac{\pi}{8} \log 2\)

Question 6
4 Marks

Evaluate: \(\int_0^4 |x - 1| dx\)

Solution:

\(\int_0^4 |x - 1| dx = \int_0^1 (1 - x) dx + \int_1^4 (x - 1) dx\)

= \([x - \frac{x^2}{2}]_0^1 + [\frac{x^2}{2} - x]_1^4\)

= \((1 - \frac{1}{2}) + [(8 - 4) - (\frac{1}{2} - 1)]\)

= \(\frac{1}{2} + [4 - (-\frac{1}{2})] = \frac{1}{2} + 4 + \frac{1}{2} = 5\)
