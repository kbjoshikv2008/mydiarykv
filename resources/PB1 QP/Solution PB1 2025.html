Value of \( \cos \left[ \frac{\pi}{6} + \cos^{-1} \left( -\frac{1}{2} \right) \right] = ... \)
(a) \( -\frac{\sqrt{3}}{2} \)
(b) \( \frac{\sqrt{3}-1}{2\sqrt{2}} \)
(c) \( \frac{\sqrt{5}-1}{4} \)
(d) \( \frac{\sqrt{3}+1}{2\sqrt{2}} \)
Correct Answer: (a) \( -\frac{\sqrt{3}}{2} \)
Explanation:
We know that \( \cos^{-1}(-x) = \pi - \cos^{-1}(x) \) for \( x \in [0,1] \).
So, \( \cos^{-1}(-1/2) = \pi - \cos^{-1}(1/2) = \pi - \pi/3 = 2\pi/3 \).
Now, \( \cos[\pi/6 + 2\pi/3] = \cos[\pi/6 + 4\pi/6] = \cos(5\pi/6) = -\cos(\pi/6) = -\sqrt{3}/2 \).
Therefore, the correct answer is (a) \( -\frac{\sqrt{3}}{2} \).

If \( A = \begin{pmatrix} 0 & 1 & c \\ -1 & a & -b \\ 2 & 3 & 0 \end{pmatrix} \) is a skew symmetric matrix, then the value of a+b + c is:
(a) 1
(b) 2
(c) 3
(d) 4
Correct Answer: (a) 1
Explanation:
For a skew-symmetric matrix, \( A^T = -A \).
Also, all diagonal elements must be zero.
So, a = 0 (from position (2,2)).

Writing the transpose: \( A^T = \begin{pmatrix} 0 & -1 & 2 \\ 1 & a & 3 \\ c & -b & 0 \end{pmatrix} \)

Setting \( A^T = -A \):
\( \begin{pmatrix} 0 & -1 & 2 \\ 1 & a & 3 \\ c & -b & 0 \end{pmatrix} = \begin{pmatrix} 0 & -1 & -c \\ 1 & -a & b \\ -2 & -3 & 0 \end{pmatrix} \)

Comparing elements:
(1,3): 2 = -c ⇒ c = -2
(2,2): a = -a ⇒ 2a = 0 ⇒ a = 0
(3,1): c = -2 ⇒ -2 = -2 (consistent)
(3,2): -b = -3 ⇒ b = 3

So a + b + c = 0 + 3 + (-2) = 1

Find 'a' and 'b', if the function given by \( f(x) = \begin{cases} ax^2 + b, & \text{if } x < 1 \\ 2x + 1, & \text{if } x \geq 1 \end{cases} \) is differentiable at \( x=1 \)
Correct Answer: a = 2, b = 1
Explanation:
For differentiability at x=1, the function must be continuous and the left and right derivatives must be equal.

Continuity: \( \lim_{x \to 1^-} f(x) = \lim_{x \to 1^+} f(x) = f(1) \)
\( a(1)^2 + b = 2(1) + 1 \) ⇒ \( a + b = 3 \) ...(1)

Differentiability: Left derivative = \( \lim_{h \to 0^-} \frac{f(1+h)-f(1)}{h} = \lim_{h \to 0^-} \frac{a(1+h)^2 + b - 3}{h} \)
= \( \lim_{h \to 0^-} \frac{a(1+2h+h^2) + b - 3}{h} = \lim_{h \to 0^-} \frac{a + 2ah + ah^2 + b - 3}{h} \)
Using (1): a+b=3, so numerator = \( 2ah + ah^2 \)
Left derivative = \( \lim_{h \to 0^-} \frac{2ah + ah^2}{h} = \lim_{h \to 0^-} (2a + ah) = 2a \)

Right derivative = \( \lim_{h \to 0^+} \frac{f(1+h)-f(1)}{h} = \lim_{h \to 0^+} \frac{2(1+h)+1 - 3}{h} = \lim_{h \to 0^+} \frac{2+2h+1-3}{h} = \lim_{h \to 0^+} \frac{2h}{h} = 2 \)

Equating: 2a = 2 ⇒ a = 1
From (1): 1 + b = 3 ⇒ b = 2
So a = 1, b = 2

The reliability of a COVID PCR test is specified as follows: Of people having COVID, 90% of the test detects the disease but 10% goes undetected. Of people free of COVID, 99% of the test is judged COVID negative but 1% are diagnosed as showing COVID positive. From a large population of which only 0.1% have COVID, one person is selected at random, given the COVID PCR test, and the pathologist reports him/her as COVID positive.

Based on the above information, answer the following:
(a) What is the probability of the 'person to be tested as COVID positive' given that 'he is actually having COVID?
(b) What is the probability of the 'person to be tested as COVID positive' given that 'he is actually not having COVID'?
(c) What is the probability that the 'person is actually not having COVID?
(d) What is the probability that the 'person is actually having COVID given that 'he is tested as COVID positive?
Correct Answer:
(a) 0.9
(b) 0.01
(c) 0.999
(d) ≈ 0.0826
Explanation:
Let C = person has COVID, T⁺ = test positive
Given: P(C) = 0.001, P(not C) = 0.999
P(T⁺|C) = 0.9, P(T⁻|C) = 0.1
P(T⁻|not C) = 0.99, P(T⁺|not C) = 0.01

(a) P(T⁺|C) = 0.9
(b) P(T⁺|not C) = 0.01
(c) P(not C) = 0.999
(d) P(C|T⁺) = ?
Using Bayes theorem:
P(C|T⁺) = P(T⁺|C)P(C) / [P(T⁺|C)P(C) + P(T⁺|not C)P(not C)]
= (0.9 × 0.001) / [(0.9 × 0.001) + (0.01 × 0.999)]
= 0.0009 / (0.0009 + 0.00999) = 0.0009 / 0.01089 ≈ 0.0826
