🔐 ANSWER KEY – Set A & Set B (Decimals)

✅ SET A SOLUTIONS

  • 1. \(3.45\) (Three and forty-five hundredths = \(3 + 0.45 = 3.45\))
  • 2. \(2350\ \text{mm} = \frac{2350}{1000}\ \text{m} = 2.350\ \text{m}\) → \(2.35\ \text{m}\)
  • 3. \(0.8 = 0.80\) (They are equal; adding zero after decimal does not change value)
  • 4. \(6.45 + 2.3 = 6.45 + 2.30 = 8.75\)
  • 5. Align decimals: \(2.75 + 1.30 = 4.05\)
    Steps: \(2.75 + 1.30 = (2 + 1) + (0.75 + 0.30) = 3 + 1.05 = 4.05\).
    Or column addition: ones: \(2+1=3\), tenths: \(7+3=10\) tenths = 1 one + 0 tenths, carry 1 → total ones 4, hundredths: 5 → \(4.05\).
  • 6. \(12.5 - 4.25 = 12.50 - 4.25\)
    Write \(12.50\) and \(4.25\):
    Hundredths: \(0-5\) → borrow 1 tenth → 10 hundredths → \(10-5=5\)
    Tenths: after borrowing, \(4\) tenths − \(2\) tenths = \(2\) tenths.
    Ones: \(12 - 4 = 8\).
    Result: \(8.25\).
  • 7. Remaining length = total rope − cut piece = \(5.75 - 2.8\)
    = \(5.75 - 2.80 = 2.95\) metres.
    Equation: \(x = 5.75 - 2.8\) → \(x = 2.95\ \text{m}\).

✅ SET B SOLUTIONS

  • 1. \(7 + \frac{4}{10} + \frac{5}{100} = 7 + 0.4 + 0.05 = 7.45\).
  • 2. \(125\ \text{cm} = \frac{125}{100}\ \text{m} = 1.25\ \text{m}\).
  • 3. \(1.09 > 1.009\) because in the tenths place: \(1.09\) has 0 tenths? Actually compare digit by digit: \(1.09\) (tenths=0, hundredths=9) vs \(1.009\) (tenths=0, hundredths=0, thousandths=9). At the hundredths place, \(9>0\) so \(1.09 > 1.009\).
  • 4. \(9.8 - 3.5 = 6.3\).
  • 5. \(10.05 - 6.875\) → write as \(10.050 - 6.875\):
    Thousandths: \(0-5\) → borrow 1 hundredth (10 thousandths) → \(10-5=5\) thousandths.
    Hundredths: after borrowing, \(4\) hundredths − \(7\) hundredths → borrow 1 tenth (10 hundredths) → \(14-7=7\) hundredths.
    Tenths: after borrowing, \(9\) tenths − \(8\) tenths = \(1\) tenth.
    Ones: \(9 - 6 = 3\).
    Result: \(3.175\).
  • 6. Total weight = \(2.5\ \text{kg} + 1.75\ \text{kg} = 2.50 + 1.75\)
    Hundredths: \(0+5=5\); tenths: \(5+7=12\) tenths = 1 one + 2 tenths (carry 1); ones: \(2+1+1=4\).
    Final: \(4.25\ \text{kg}\).
  • 7. Distance from 4:
    \(|4 - 3.95| = 0.05\)
    \(|4 - 4.12| = 0.12\)
    Since \(0.05 < 0.12\), \(3.95\) is closer to \(4\). Reasoning: on the number line, 3.95 is only 0.05 units left of 4, while 4.12 is 0.12 units right of 4.
