🔐 ANSWER KEY – Set A & Set B

✅ SET A ANSWERS

  • 1. (1 mark) One lakh has 5 zeros. Numeral: 1,00,000.
  • 2. (1 mark) Ten lakhs two hundred thirty five = 10,00,235.
  • 3. (1 mark) 90,000 ÷ 1000 = 90 times pressing +1000 button.
  • 4. (1 mark) Number 3,87,69,957 → nearest thousand: look at last 3 digits 957 → 957 > 500, so round up thousand part: 3,87,70,000. Answer: 3,87,70,000.
  • 5. (2 marks) Two different ways for 40,629:
     ✔ Way 1: 4×10000 + 0×1000 + 6×100 + 2×10 + 9×1 → (4 presses of +10000, 6 of +100, 2 of +10, 9 of +1) total 21 clicks.
     ✔ Way 2: 40×1000 + 6×100 + 2×10 + 9×1 → 40 presses +1000, 6 +100, 2 +10, 9 +1 → total 57 clicks.
    (Any two distinct valid decompositions accepted, e.g., 406×100 + 29×1 etc.)
  • 6. (2 marks) Exact increase = 44,67,797 − 24,33,835 = 20,33,962.
    (a) Rounded to nearest lakh: 20,33,962 → lakhs digit 33,962 < 50,000 → 20,00,000 (20 lakh).
    (b) Words: Twenty lakh thirty-three thousand nine hundred sixty-two.
  • 7. (2 marks) Exact difference = 14,63,128 − 4,90,020 = 9,73,108.
    (a) Roxie: diff from 10,00,000 = 26,892; Estu: diff from 9,00,000 = 73,108 → Roxie closer.
    (b) Exact difference = 9,73,108.
    (c) 9,73,108 is greater than 9,50,000 because 9,73,108 − 9,50,000 = 23,108 > 0.

✅ SET B ANSWERS

  • 1. (1 mark) One crore has 7 zeros. Numeral: 1,00,00,000.
  • 2. (1 mark) Four lakh seven thousand seven hundred four = 4,07,704.
  • 3. (1 mark) 53,000 ÷ 100 = 530 times pressing +100 button.
  • 4. (1 mark) 29,05,32,481 → nearest crore: look at the crores digit and next digits: 29,05,32,481 → 05,32,481 < 50,00,000? Actually 5,32,481 < 50 lakh? Wait, standard rounding: 29,05,32,481, the next part after 29 crore is 5,32,481 which is less than 50,00,000? No, 5,32,481 is less than 50 lakh, so round down → 29,00,00,000.
  • 5. (2 marks) Two different ways for 56,354:
     ✔ Way 1: 5×10000 + 6×1000 + 3×100 + 5×10 + 4×1 → (5+6+3+5+4 = 23 clicks)
     ✔ Way 2: 56×1000 + 3×100 + 5×10 + 4×1 → 56 + 3 + 5 + 4 = 68 clicks.
    (Any valid different grouping, e.g., 563×100 + 54×1 → 563 + 54 = 617 clicks, accepted.)
  • 6. (2 marks) Exact increase = 18,18,872 − 12,62,551 = 5,56,321.
    (a) Rounded to nearest thousand: 5,56,321 → last three digits 321 < 500 → 5,56,000.
    (b) Words: Five lakh fifty-six thousand three hundred twenty-one.
  • 7. (2 marks) Exact difference = 1,24,42,373 − 1,19,78,450 = 4,63,923.
    (a) Nearest lakh: 4,63,923 → lakhs part 63,923 > 50,000 → round up to 5,00,000.
    (b) Exact difference = 4,63,923.
    (c) 4,63,923 is less than 5,00,000 because 4,63,923 < 5,00,000. Justification: 5,00,000 − 4,63,923 = 36,077 positive.
