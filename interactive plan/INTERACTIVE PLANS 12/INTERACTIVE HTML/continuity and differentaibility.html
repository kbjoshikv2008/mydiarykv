1. Continuity – Formal Definition & Examples

Definition: A function \(f\) is continuous at \(x=c\) iff \(\lim_{x\to c^-}f(x) = \lim_{x\to c^+}f(x) = f(c)\).

📘 Example 3 (NCERT): \(f(x)=|x|\) is continuous at \(x=0\).
LHL = \(\lim_{x\to0^-}(-x)=0\), RHL = \(\lim_{x\to0^+}x=0\), \(f(0)=0\) ✅
⚠️ Example 4: \(f(x)=\begin{cases} x^3+3, & x\neq0 \\ 1, & x=0 \end{cases}\) is discontinuous at 0 because \(\lim_{x\to0}f(x)=3 \neq f(0)\).
Greatest integer function: \(f(x)=[x]\) is discontinuous at every integer. At \(x=1\): LHL=0, RHL=1, value=1 → not equal.

2. Differentiability & Theorem 3 (Differentiable ⇒ Continuous)

Derivative: \(f'(c)=\lim_{h\to0}\frac{f(c+h)-f(c)}{h}\) (if exists).

📖 Theorem: If \(f\) is differentiable at \(c\) ⇒ \(f\) continuous at \(c\). Converse false: \(|x|\) continuous but not differentiable at 0 (LHD = -1, RHD = 1).

✍️ Quick check: Which function is NOT differentiable at \(x=1\)?

\(f(x)=x^2\)
\(f(x)=|x-1|\)
\(f(x)=3x+2\)
Geometric meaning: Differentiable ⇒ graph has a unique tangent (no corner, no cusp).
Example from NCERT: Show that \(f(x)=|x-1|, x\in\mathbb{R}\) is not differentiable at \(x=1\). LHD = -1, RHD = 1.
3. Derivative Techniques (Chain, Implicit, Log Differentiation)

🔗 Chain Rule

\(\frac{d}{dx}[f(g(x))] = f'(g(x))\cdot g'(x)\).
Example: \(\frac{d}{dx}\sin(x^2) = \cos(x^2)\cdot 2x\).

Let \(u=x^2\), \(\frac{dy}{du}=\cos u\), \(\frac{du}{dx}=2x\) ⇒ derivative = \(2x\cos(x^2)\).

📌 Implicit Differentiation

\(x^2 + y^2 = 25\) ⇒ \(2x + 2y\frac{dy}{dx}=0\) ⇒ \(\frac{dy}{dx}=-\frac{x}{y}\).

📈 Inverse Trig Derivatives

\(\frac{d}{dx}\sin^{-1}x = \frac{1}{\sqrt{1-x^2}}\), \(\frac{d}{dx}\tan^{-1}x = \frac{1}{1+x^2}\).

🧪 Logarithmic Differentiation (for \(y=[u(x)]^{v(x)}\))

Example: \(y = x^{\sin x}\) ⇒ \(\ln y = \sin x \ln x\) ⇒ \(\frac{dy}{dx}=y\left(\cos x \ln x + \frac{\sin x}{x}\right)\).

\(\frac{dy}{dx} = x^{\sin x}\left(\cos x \ln x + \frac{\sin x}{x}\right)\).
4. Parametric & Second Order Derivatives

If \(x = f(t), y = g(t)\) then \(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\). Second order: \(\frac{d^2y}{dx^2} = \frac{d}{dt}\left(\frac{dy}{dx}\right)/\frac{dx}{dt}\).

NCERT Example 33: \(x = a(\theta+\sin\theta), y = a(1-\cos\theta)\) ⇒ \(\frac{dy}{dx} = \tan\frac{\theta}{2}\).
\(\frac{d^2y}{dx^2} = \frac{\sec^2(\theta/2)\cdot\frac12}{a(1+\cos\theta)} = \frac{1}{2a(1+\cos\theta)}\sec^2(\theta/2)\) simplifies to \(\frac{1}{4a\cos^4(\theta/2)}\).
5. Important Questions from NCERT Exercises (Board Exam Perspective)
Q1 (Ex 5.1, Q.14): Discuss continuity of \(f(x)=\begin{cases}3, & 0\le x\le 1\\4, & 1
👉 Check at \(x=1\): LHL = 3, RHL = 4 → not continuous. At \(x=3\): LHL = 4, RHL = 5 → not continuous. So points of discontinuity: \(x=1\) and \(x=3\).
Q2 (Ex 5.1, Q.18): For what value of \(\lambda\) is \(f(x)=\begin{cases}\lambda(x^2-2x), & x\le0\\4x+1, & x>0\end{cases}\) continuous at \(x=0\)?
Continuity at 0: LHL = \(\lambda(0-0)=0\), RHL = \(4(0)+1=1\), so \(0=1\) impossible → no such \(\lambda\) exists. So cannot be made continuous at 0.
Q3 (Ex 5.3, Q.9): Find \(\frac{dy}{dx}\) if \(y = \sin^{-1}\left(\frac{2x}{1+x^2}\right)\)
Let \(x=\tan\theta\) ⇒ \(\frac{2x}{1+x^2} = \sin2\theta\) ⇒ \(y = \sin^{-1}(\sin2\theta) = 2\theta = 2\tan^{-1}x\) ⇒ \(\frac{dy}{dx} = \frac{2}{1+x^2}\) for \(|x|<1\).
Q4 (Ex 5.5, Q.5): Differentiate \((x+3)^2 (x+4)^3 (x+5)^4\) w.r.t x using logarithmic differentiation.
Let \(y = (x+3)^2 (x+4)^3 (x+5)^4\). \(\ln y = 2\ln(x+3)+3\ln(x+4)+4\ln(x+5)\). Diff: \(\frac{1}{y}\frac{dy}{dx} = \frac{2}{x+3}+\frac{3}{x+4}+\frac{4}{x+5}\). Thus \(\frac{dy}{dx} = y\left(\frac{2}{x+3}+\frac{3}{x+4}+\frac{4}{x+5}\right)\).
Q5 (Ex 5.7, Q.11): If \(y = 5\cos x - 3\sin x\), prove that \(\frac{d^2y}{dx^2}+y=0\).
\(\frac{dy}{dx} = -5\sin x - 3\cos x\). \(\frac{d^2y}{dx^2} = -5\cos x + 3\sin x = -(5\cos x - 3\sin x) = -y\). Hence \(\frac{d^2y}{dx^2}+y=0\).
Q6 (Miscellaneous, Q.14): If \(x\sqrt{1+y}+y\sqrt{1+x}=0\) for \(-1
Given \(x\sqrt{1+y} = -y\sqrt{1+x}\). Square both sides: \(x^2(1+y) = y^2(1+x)\) ⇒ \(x^2 + x^2y = y^2 + xy^2\) ⇒ Rearr: \(x^2 - y^2 + xy(x - y)=0\) ⇒ \((x-y)(x+y+xy)=0\). Since \(x\neq y\) (check domain), we have \(x+y+xy=0\) ⇒ \(y(1+x) = -x\) ⇒ \(y = -\frac{x}{1+x}\). Differentiate: \(\frac{dy}{dx} = -\frac{(1+x)-x}{(1+x)^2} = -\frac{1}{(1+x)^2}\).
Q7 (Ex 5.3, Q.14): Find \(\frac{dy}{dx}\) if \(y = \sin^{-1}\left(2x\sqrt{1-x^2}\right)\), \(-\frac{1}{\sqrt{2}}
Let \(x=\sin\theta\) ⇒ \(2x\sqrt{1-x^2} = 2\sin\theta\cos\theta = \sin2\theta\) ⇒ \(y = \sin^{-1}(\sin2\theta) = 2\theta = 2\sin^{-1}x\) ⇒ \(\frac{dy}{dx} = \frac{2}{\sqrt{1-x^2}}\).
Q8 (Example 30, page 132): Find \(\frac{dy}{dx}\) if \(y^x + x^y + x^x = a^b\).
Let \(u=y^x, v=x^y, w=x^x\). Logarithmic diff yields: \(\frac{du}{dx}=y^x\left(\frac{x}{y}\frac{dy}{dx}+\ln y\right)\), \(\frac{dv}{dx}=x^y\left(\frac{y}{x}+\ln x\frac{dy}{dx}\right)\), \(\frac{dw}{dx}=x^x(1+\ln x)\). Sum zero → solve for \(\frac{dy}{dx}\). Final: \(\frac{dy}{dx} = -\frac{y^x\ln y + y x^{y-1} + x^x(1+\ln x)}{x y^{x-1} + x^y \ln x}\).
Tip: Practice these questions thoroughly – they frequently appear in board exams and competitive tests.
