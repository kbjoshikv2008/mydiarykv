1. Sets & Their Representations

Set: a well-defined collection of distinct objects. Denoted by capital letters, elements by ∈.

Roster Form

Listing elements inside braces { }.
Example: Vowels = {a, e, i, o, u}
Natural numbers less than 6 = {1,2,3,4,5}

Set-Builder Form

Describing common property: {x : property}
Example: {x : x is a vowel in English alphabet}
{x | x ∈ ℕ, x<6}

3. Equal Sets & Subsets

Equal Sets: exactly same elements. Subset (A ⊂ B): every element of A is in B. Proper subset: A ⊂ B and A ≠ B. Every set is subset of itself, ∅ is subset of every set.

Example: {1,2,3} = {3,1,2}.
{a,b,c} not equal to {a,b,d}.

If A = {1,2} and B = {1,2,3}, then A ⊂ B.

4. Intervals as Subsets of ℝ

5. Venn Diagrams

Universal set (rectangle) and subsets (circles). Visual representation of union, intersection, difference.

Union (A ∪ B) = total shaded, Intersection (A ∩ B) = overlapping region.

6. Set Operations: ∪ , ∩ , −

Union A ∪ B = {x: x∈A or x∈B}. Intersection A ∩ B = {x: x∈A and x∈B}. Difference A - B = {x∈A : x∉B}.

7. Complement & De Morgan’s Laws

Complement: A' = U − A = {x∈U : x∉A}. De Morgan's Laws: (A ∪ B)' = A' ∩ B'   and   (A ∩ B)' = A' ∪ B'.
